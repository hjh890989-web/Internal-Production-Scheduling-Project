### `LineCompatibilityRule.java`

```java
@Component @RequiredArgsConstructor
public class LineCompatibilityRule {

    private final LineCompatibilityRepository compatRepo;

    /**
     * BR-E08: 포드 전용 품번 → 신규 라인 차단.
     *  - is_compatible=FALSE row 존재 → fail
     *  - row 없음 → 기본 호환 (pass)
     */
    public RuleResult evaluate(String hoseId, String lineId) {
        var record = compatRepo.findByHoseIdAndLineId(hoseId, lineId);
        if (record.isPresent() && !record.get().getIsCompatible()) {
            return RuleResult.fail("BR-E08 라인 호환성 위반: %s × %s (라인 호환 안 됨)"
                .formatted(hoseId, lineId));
        }
        return RuleResult.pass();
    }
}
```
