### `Auditable.java`

```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Auditable {
    /**
     * Reason 표현식 — SpEL 지원. 예: "#scheduleId + ' confirm'"
     */
    String reason() default "";
}
```

### `AuditableAspect.java`

```java
@Aspect @Component @RequiredArgsConstructor @Slf4j
public class AuditableAspect {

    private final JdbcTemplate jdbc;
    private final SecurityContextService securityCtx;
    private final SpelExpressionParser spelParser = new SpelExpressionParser();

    @Around("@annotation(auditable)")
    public Object setAuditContext(ProceedingJoinPoint pjp, Auditable auditable) throws Throwable {
        String actor = securityCtx.currentUserId();      // Keycloak token → user ID
        String reason = resolveReason(auditable.reason(), pjp);

        // PostgreSQL session var — transaction scope
        jdbc.execute(String.format("SET LOCAL audit.actor = '%s'", escape(actor)));
        if (reason != null && !reason.isBlank()) {
            jdbc.execute(String.format("SET LOCAL audit.reason = '%s'", escape(reason)));
        }

        try {
            return pjp.proceed();
        } finally {
            // SET LOCAL은 transaction 종료 시 자동 해제 (별도 RESET 불필요)
        }
    }

    private String resolveReason(String template, ProceedingJoinPoint pjp) {
        if (template.isBlank()) return null;
        if (!template.startsWith("#")) return template;       // 정적 문자열

        // SpEL 평가
        var ctx = new StandardEvaluationContext();
        var sig = (MethodSignature) pjp.getSignature();
        String[] names = sig.getParameterNames();
        Object[] args = pjp.getArgs();
        for (int i = 0; i < names.length; i++) ctx.setVariable(names[i], args[i]);

        try {
            return spelParser.parseExpression(template).getValue(ctx, String.class);
        } catch (Exception e) {
            log.warn("@Auditable reason SpEL 실패: {}", template, e);
            return template;
        }
    }

    private String escape(String s) {
        return s == null ? "system" : s.replace("'", "''");
    }
}
```

## :shield: Risk Mitigation

- SpEL 평가 실패 — 정적 fallback (warn log)
- SET LOCAL injection — escape 단순 ('→''), 향후 bind parameter 검토 (현재는 admin-controlled annotation value)
- multi-thread isolation — Spring transaction context 의존 (ThreadLocal 추가 안전망)

## :memo: Implementation Notes

- SpEL은 메서드 파라미터 (`#scheduleId`)·인스턴스 필드 (`#root.someField`) 모두 지원
- `SET LOCAL`은 transaction commit 시 자동 reset — leak 없음
- 향후 attribute-level diff 표시 검토 — 현재는 JSONB full row
