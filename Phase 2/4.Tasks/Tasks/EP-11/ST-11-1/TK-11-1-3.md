### 핵심 로직

```java
@SpringBootTest @Testcontainers
class AuditCoverageRegressionIT {

    @Autowired VcScheduleRepository vcRepo;
    @Autowired ExScheduleRepository exRepo;
    @Autowired OrderRepository orderRepo;
    @Autowired AuditLogRepository auditRepo;
    @Autowired VcScheduleConfirmService confirmService;

    @Test
    void vc_schedule_insert_creates_one_audit_row() {
        long before = auditRepo.countVcLogs();
        vcRepo.save(buildDraft());
        long after = auditRepo.countVcLogs();

        assertThat(after - before).isEqualTo(1);
        var latest = auditRepo.findLatestVcLog();
        assertThat(latest.getOperation()).isEqualTo("INSERT");
        assertThat(latest.getOldRow()).isNull();
        assertThat(latest.getNewRow()).isNotNull();
    }

    @Test
    void confirm_creates_update_audit_with_actor_and_reason() {
        var schedule = createCandidate();

        withPlannerRole("planner1", () -> confirmService.confirm(schedule.getId()));

        var latest = auditRepo.findLatestVcLogForTarget(schedule.getId());
        assertThat(latest.getOperation()).isEqualTo("UPDATE");
        assertThat(latest.getActor()).isEqualTo("planner1");
        assertThat(latest.getReason()).contains("confirm vc schedule");
    }

    @Test
    void direct_sql_update_via_app_user_blocked_or_audited() throws SQLException {
        // app_user는 status 직접 UPDATE 차단 (EP-10) — 본 테스트는 별도 컬럼 UPDATE
        var schedule = createDraft();

        try (Connection conn = openAsAppUser()) {
            try (PreparedStatement ps = conn.prepareStatement(
                "UPDATE vc_schedule SET production_date = production_date + INTERVAL '1 day' WHERE id = ?")) {
                ps.setObject(1, schedule.getId());
                ps.executeUpdate();
            }
        }

        // AOP context 없음 → actor='system' fallback
        var latest = auditRepo.findLatestVcLogForTarget(schedule.getId());
        assertThat(latest.getOperation()).isEqualTo("UPDATE");
        assertThat(latest.getActor()).isEqualTo("system");
    }

    @Test
    void week_horizon_all_changes_audited_one_to_one() {
        // 1주 분량 시나리오 — VC·EX·Order 변경 ≈ 1,000 건
        var stats = runWeekHorizonScenario();

        assertThat(auditRepo.countVcLogs()).isEqualTo(stats.vcChanges());
        assertThat(auditRepo.countExLogs()).isEqualTo(stats.exChanges());
        assertThat(auditRepo.countOrderLogs()).isEqualTo(stats.orderChanges());
    }

    @Test
    void delete_creates_audit_with_old_row_only() {
        var schedule = createDraft();
        UUID id = schedule.getId();

        vcRepo.deleteById(id);

        var latest = auditRepo.findLatestVcLogForTarget(id);
        assertThat(latest.getOperation()).isEqualTo("DELETE");
        assertThat(latest.getOldRow()).isNotNull();
        assertThat(latest.getNewRow()).isNull();
    }

    @Test
    void multi_thread_actor_isolation() throws InterruptedException {
        var threadA = new Thread(() -> withPlannerRole("plannerA",
            () -> vcRepo.save(buildDraft())));
        var threadB = new Thread(() -> withPlannerRole("plannerB",
            () -> vcRepo.save(buildDraft())));

        threadA.start(); threadB.start();
        threadA.join(); threadB.join();

        var logs = auditRepo.findRecentVcLogs(2);
        var actors = logs.stream().map(AuditLog::getActor).collect(Collectors.toSet());
        assertThat(actors).containsExactlyInAnyOrder("plannerA", "plannerB");
    }
}
```

## :test_tube: Acceptance Criteria

**검증**: T-I + A

- [ ] **INSERT → audit row 1건** (operation='INSERT', new_row 채움)
- [ ] **UPDATE → audit row** (old/new 모두 채움)
- [ ] **DELETE → audit row** (operation='DELETE', new_row NULL)
- [ ] **@Auditable confirm → actor='planner1', reason 포함**
- [ ] **AOP 미적용 변경 → actor='system'**
- [ ] **1주 호라이즌 회귀** — VC·EX·Order 변경 수 = audit row 수 (1:1)
- [ ] **multi-thread isolation** — 서로 다른 actor 분리
- [ ] CI Jenkins stage 통합
- [ ] 통합 테스트 ≥ 6 케이스
