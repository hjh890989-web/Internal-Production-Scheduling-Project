### `postgresql.conf` 추가

```
archive_mode = on
archive_timeout = 300                            # 5분마다 강제 switch
archive_command = '/usr/local/bin/archive-wal.sh %p %f'
max_wal_senders = 3
wal_level = replica
```

### `archive-wal.sh`

```bash
#!/bin/bash
set -e
WAL_PATH=$1
WAL_NAME=$2

# S3 upload (parallel, 빠른 응답)
aws s3 cp "${WAL_PATH}" "s3://${BACKUP_BUCKET}/wal/${WAL_NAME}" \
  --quiet --storage-class STANDARD_IA

# 실패 시 exit 1 → PostgreSQL이 재시도
```
