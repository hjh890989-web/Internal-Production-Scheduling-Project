### 핵심 로직 / 알고리즘

1. **엑셀 로드 (한글 인코딩 안전)**: `openpyxl.load_workbook()`로 `성형공정_제약조건.xlsx` 열기. 시트명 = `Sheet1 (2)` (현재 마스터). UTF-8 강제 출력.

2. **컬럼 위치 확정**: row 2의 헤더에서 K열·L열이 각각 `좌측셋팅`·`우측셋팅`인지 확인. 불일치 시 즉시 fail (마스터 구조 변경 감지).

3. **47품번 순회 검증** (row 3~49):
   - K열 값 ∈ {`o`, `x`} → 통과 / 그 외(`O`, `X`, NULL, 공백, 한글 등) → 위반
   - L열 동일 검증
   - 추가 검증 (선택): `(K='x', L='x')` 즉 좌·우 모두 불가인 품번 존재 여부 → 경고 출력 (저압 슬롯 사용 불가 의미하므로 IC 전용 또는 스케줄 불가 품번 의심)

4. **결과 리포트 생성**: 마크다운 표 형식. 컬럼 = HOSE | K | L | K_valid | L_valid | 비고. 위반 행은 ⚠️ 표시.

5. **종료 코드**: 위반 0건 → exit 0 / 1건 이상 → exit 1 (CI/CD 통합 대비).

### 예시 코드

```python
#!/usr/bin/env python3
# tools/master_validation/validate_vc_master_kl.py
import sys
import io
from datetime import date
from pathlib import Path
import openpyxl

# Korean 인코딩 안전 (Windows cp949 회피)
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8')

VALID_VALUES = {'o', 'x'}
MASTER_PATH = Path('Phase 1/2.Raw Materials/Vulcanization/성형공정_제약조건.xlsx')
REPORT_DIR = Path('tools/master_validation/reports')

def validate_kl(wb):
    ws = wb['Sheet1 (2)']
    # 헤더 row 2: K열 헤더 = '좌측셋팅', L열 헤더 = '우측셋팅'
    assert ws.cell(row=2, column=11).value == '좌측셋팅', "K열 헤더 불일치"
    assert ws.cell(row=2, column=12).value == '우측셋팅', "L열 헤더 불일치"

    violations = []
    rows = []
    for r in range(3, ws.max_row + 1):
        hose = ws.cell(row=r, column=1).value
        if not hose:
            break
        k = ws.cell(row=r, column=11).value
        l = ws.cell(row=r, column=12).value
        k_valid = k in VALID_VALUES
        l_valid = l in VALID_VALUES
        warn = ''
        if k == 'x' and l == 'x':
            warn = '좌·우 모두 불가 — 저압 슬롯 사용 불가 (IC 전용 또는 스케줄 불가 의심)'
        rows.append((hose, k, l, k_valid, l_valid, warn))
        if not (k_valid and l_valid):
            violations.append((hose, k, l))
    return rows, violations

def write_report(rows, violations):
    REPORT_DIR.mkdir(parents=True, exist_ok=True)
    today = date.today().isoformat()
    out = REPORT_DIR / f'vc_master_kl_{today}.md'
    with open(out, 'w', encoding='utf-8') as f:
        f.write(f'# 성형 마스터 K/L열 검증 리포트 ({today})\n\n')
        f.write(f'- 총 품번: {len(rows)}\n')
        f.write(f'- 위반: {len(violations)}\n\n')
        f.write('| HOSE | K(좌측) | L(우측) | K_valid | L_valid | 비고 |\n')
        f.write('|---|:---:|:---:|:---:|:---:|---|\n')
        for hose, k, l, kv, lv, warn in rows:
            mark = '⚠️ ' if not (kv and lv) else ''
            f.write(f'| {mark}{hose} | {k} | {l} | {kv} | {lv} | {warn} |\n')
    return out

if __name__ == '__main__':
    wb = openpyxl.load_workbook(MASTER_PATH, data_only=True)
    rows, violations = validate_kl(wb)
    report = write_report(rows, violations)
    print(f'Report: {report}')
    print(f'Total: {len(rows)} 품번, 위반: {len(violations)}')
    sys.exit(0 if not violations else 1)
```
