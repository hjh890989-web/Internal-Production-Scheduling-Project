## :hammer_and_wrench: Implementation Plan
```
infra/nginx/
  whitelist.conf
  scheduling.conf                                [include whitelist]
```

```nginx
geo $allowed_internal {
    default 0;
    10.0.0.0/8 1;          # 사내 사설망
    172.16.0.0/12 1;
    192.168.0.0/16 1;
}

server {
    if ($allowed_internal = 0) {
        return 403;
    }
    # ... 일반 server config
}
```
