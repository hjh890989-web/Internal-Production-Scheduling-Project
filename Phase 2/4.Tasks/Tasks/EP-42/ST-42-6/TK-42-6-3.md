### Keycloak realm 필드 (실 적용)
```json
{
  "registrationEmailAsUsername": false,
  "loginWithEmailAllowed": false,
  "bruteForceProtected": true,
  "permanentLockout": false,
  "maxFailureWaitSeconds": 600,
  "failureFactor": 5,
  "passwordPolicy": "length(4) and regexPattern(^[0-9]{4}$) and notUsername(undefined)"
}
```

### 사용자 (username = 사번 8자리)
- `12345678` 형식 — 일반 사용자 (IT 부서가 입사 시 발급)
- `99000001`~`99000003` — emergency 계정 (봉인 봉투)

## :test_tube: Acceptance Criteria
- [ ] 사번 (8자리 숫자) 로그인 가능
- [ ] 이메일 로그인 불가 (HTTP 401)
- [ ] 4자리 숫자 PIN 만 허용
  - "1234" ✓ / "12345" ✗ / "abcd" ✗ / "12 4" ✗
- [ ] PIN = username 거부 (사번이 12345678인데 PIN=1234일 때 별 문제 없음 — notUsername 은 정확히 같을 때만)
- [ ] 5회 실패 → 계정 잠금
- [ ] 잠금 후 10분 경과 → 자동 해제
- [ ] Brute force 1000회 시도 시뮬 — failureFactor 5 이상 자동 차단
