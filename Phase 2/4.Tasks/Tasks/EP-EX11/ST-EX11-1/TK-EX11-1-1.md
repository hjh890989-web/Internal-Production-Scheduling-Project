### `CumulativeYieldRule.java`

```java
@Component @RequiredArgsConstructor
public class CumulativeYieldRule {

    /**
     * BR-E10: deadline 이전 (포함) 누적 yield ≥ Q_ext.
     *  - production_date ≤ deadline 행만 합산
     *  - 결과 < Q_ext → fail
     */
    public ValidationViolation validate(ExScheduleCandidate cand, ExtrusionLedger ledger) {
        long cumulative = ledger.cumulativeYield(cand.hoseId(), cand.extrusionDeadline());
        long required = cand.getQExt();

        if (cumulative >= required) return null;       // pass

        return new ValidationViolation(
            ValidationCategory.CUMULATIVE_YIELD_SHORT,
            "BR-E10 누적 yield 부족: %s deadline=%s 누적=%d < Q_ext=%d"
                .formatted(cand.hoseId(), cand.extrusionDeadline(), cumulative, required),
            cand.getId()
        );
    }
}
```

### `ExtrusionLedger.java`

```java
public class ExtrusionLedger {

    // (hose_id, production_date) → daily yield
    private final NavigableMap<Tuple2<String, LocalDate>, Long> dailyYield = new TreeMap<>();

    public void addYield(String hoseId, LocalDate date, long yield) {
        dailyYield.merge(Tuple.of(hoseId, date), yield, Long::sum);
    }

    /**
     * deadline 이전 (포함) 누적 합산.
     */
    public long cumulativeYield(String hoseId, LocalDate deadline) {
        long sum = 0L;
        for (var entry : dailyYield.entrySet()) {
            var key = entry.getKey();
            if (!key.first().equals(hoseId)) continue;
            if (key.second().isAfter(deadline)) continue;
            sum += entry.getValue();
        }
        return sum;
    }

    public static ExtrusionLedger fromAllocationResult(ExAllocationResult result) {
        var ledger = new ExtrusionLedger();
        for (var row : result.schedule()) {
            ledger.addYield(row.hoseId(), row.shiftSlot().date(), row.actualYield());
        }
        return ledger;
    }
}
```

### `ValidationResult` + `ValidationViolation`

```java
public record ValidationResult(
    UUID candidateId,
    boolean passed,
    List<ValidationViolation> violations,
    Instant measuredAt
) {
    public static ValidationResult pass(UUID id) {
        return new ValidationResult(id, true, List.of(), Instant.now());
    }
    public static ValidationResult fail(UUID id, List<ValidationViolation> v) {
        return new ValidationResult(id, false, v, Instant.now());
    }
}

public record ValidationViolation(ValidationCategory category, String reason, UUID candidateId) {}

public enum ValidationCategory {
    CUMULATIVE_YIELD_SHORT("BR-E10", "누적 yield 부족"),
    SHIFT_CAPACITY_EXCEEDED("BR-E04", "shift 용량 초과");

    private final String brCode;
    private final String description;
}
```
