### `ShiftCapacityRule.java`

```java
@Component @RequiredArgsConstructor
public class ShiftCapacityRule {

    private final ShiftEfficiencyService shiftService;

    /**
     * BR-E04: 누적 actualMin ≤ effective_min (shift당 180).
     */
    public ValidationViolation validate(ShiftSlot slot, ExtrusionLedger ledger) {
        int cumulativeMin = ledger.minutesForSlot(slot);
        int effectiveMin = shiftService.effectiveMin(slot.shiftCode());

        if (cumulativeMin <= effectiveMin) return null;

        return new ValidationViolation(
            ValidationCategory.SHIFT_CAPACITY_EXCEEDED,
            "BR-E04 shift 용량 초과: %s 누적=%d > effective=%d"
                .formatted(slot, cumulativeMin, effectiveMin),
            null
        );
    }
}
```

### `ExtrusionLedger.java` 수정

```java
public class ExtrusionLedger {

    // 기존 dailyYield + 추가
    private final Map<ShiftSlot, Integer> minutesPerSlot = new HashMap<>();

    public void addMinutes(ShiftSlot slot, int min) {
        minutesPerSlot.merge(slot, min, Integer::sum);
    }

    public int minutesForSlot(ShiftSlot slot) {
        return minutesPerSlot.getOrDefault(slot, 0);
    }

    /**
     * 모든 slot 순회 — Validation Gate가 각 slot 별 capacity 검증 시 사용.
     */
    public Set<ShiftSlot> allSlots() {
        return Set.copyOf(minutesPerSlot.keySet());
    }

    public static ExtrusionLedger fromAllocationResult(ExAllocationResult result) {
        var ledger = new ExtrusionLedger();
        for (var row : result.schedule()) {
            ledger.addYield(row.hoseId(), row.shiftSlot().date(), row.actualYield());
            ledger.addMinutes(row.shiftSlot(), row.actualMin());   // ← 추가
        }
        return ledger;
    }
}
```

### `ExtrusionValidationGate.java`

```java
@Service @RequiredArgsConstructor
public class ExtrusionValidationGate {

    private final CumulativeYieldRule yieldRule;
    private final ShiftCapacityRule capacityRule;

    /**
     * 후보당 검증 — 2 Rule 통합.
     *  1. 누적 yield ≥ Q_ext (deadline 이전)
     *  2. shift capacity ≤ effective_min
     */
    public ValidationResult validate(ExScheduleCandidate cand, ExtrusionLedger ledger) {
        var violations = new ArrayList<ValidationViolation>();

        var yv = yieldRule.validate(cand, ledger);
        if (yv != null) violations.add(yv);

        // shift capacity는 후보 위치한 모든 slot 검증
        for (var slot : ledger.slotsForHose(cand.hoseId())) {
            var cv = capacityRule.validate(slot, ledger);
            if (cv != null) violations.add(cv);
        }

        return violations.isEmpty()
            ? ValidationResult.pass(cand.getId())
            : ValidationResult.fail(cand.getId(), violations);
    }

    /**
     * Batch — 모든 후보 일괄 검증 (TK-EX11-1-3 성능 측정 대상).
     */
    public List<ValidationResult> validateAll(List<ExScheduleCandidate> candidates,
                                                ExtrusionLedger ledger) {
        return candidates.parallelStream()
            .map(c -> validate(c, ledger))
            .toList();
    }
}
```

## :test_tube: Acceptance Criteria

**검증**: T-U + T-I

- [ ] **누적 min ≤ 180** → pass
- [ ] **누적 min = 181** → fail (SHIFT_CAPACITY_EXCEEDED)
- [ ] **빈 slot** → cumulative = 0 → pass
- [ ] **다른 효율 shift** (efficiency 0.80 → 192) — 192 통과
- [ ] **2 Rule 동시 위반** → violations 2건 (yield + capacity)
- [ ] **`validateAll` parallel** — 단일 thread 대비 ≥ 2x
- [ ] 단위 + 통합 테스트 ≥ 5 케이스
