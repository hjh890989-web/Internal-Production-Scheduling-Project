### `postgresql.conf`

```
archive_mode = on
archive_timeout = 300                            # 5분마다 강제 switch
archive_command = '/usr/local/bin/archive-wal.sh %p %f'
```

### `archive-wal.sh`

```bash
#!/bin/bash
WAL_PATH=$1
WAL_NAME=$2
aws s3 cp "${WAL_PATH}" "s3://${BACKUP_BUCKET}/wal/${WAL_NAME}" --quiet
```
