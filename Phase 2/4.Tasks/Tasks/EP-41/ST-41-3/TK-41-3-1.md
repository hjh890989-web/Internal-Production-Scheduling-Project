### `MesChaosIT.java`

```java
@SpringBootTest @Testcontainers
class MesChaosIT {

    @Autowired MesActualReceiver receiver;
    @Autowired ToxiproxyClient toxiproxy;

    @Test
    void mes_timeout_for_one_shift_triggers_fallback() {
        // MES API timeout 시뮬 (4시간)
        toxiproxy.upstream("mes").addToxic(new TimeoutToxic(60_000));

        // 1 shift 동안 미수신
        Thread.sleep(Duration.ofMinutes(5).toMillis());

        // Fallback: 임시 카운트 적용 (TK-41-3-2)
        assertThat(receiver.findLatestActual())
            .extracting(MesActual::getStatus)
            .isEqualTo(ActualStatus.FALLBACK_ESTIMATED);
    }
}
```
