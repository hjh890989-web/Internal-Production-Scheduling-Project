### 핵심 로직

```java
package com.scheduling.vc.required;

import com.scheduling.master.product.Product;
import com.scheduling.master.product.ProductRepository;
import com.scheduling.order.domain.Order;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Service;

import java.time.LocalDate;
import java.util.*;

@Service @RequiredArgsConstructor
public class RequiredQuantityCalculator {

    private final ProductRepository productRepo;

    /**
     * REQ-FUNC-VC-009: Q_required = max(0, Q_net + target_stock - current_stock)
     *
     * @param orders 호라이즌 내 수주 (이미 EP-02 중복 해소·EP-03 Diff 처리된 정본)
     * @param horizonFrom·to 영업일 호라이즌 (보통 today ~ today+7d)
     */
    public Map<String, Integer> calculate(List<Order> orders,
                                          LocalDate horizonFrom,
                                          LocalDate horizonTo) {
        // 1. 호라이즌 내 수주만 필터
        var inHorizon = orders.stream()
            .filter(o -> !o.getDeliveryDate().isBefore(horizonFrom)
                      && !o.getDeliveryDate().isAfter(horizonTo))
            .toList();

        // 2. hose_id별 Q_net 집계
        var qNetByHose = new HashMap<String, Integer>();
        for (Order o : inHorizon) {
            qNetByHose.merge(o.getHoseId(), o.getQty(), Integer::sum);
        }

        // 3. PRODUCT 마스터 조회 (target_stock, current_stock)
        var hoseIds = qNetByHose.keySet();
        var products = productRepo.findAllById(hoseIds);
        var productMap = products.stream()
            .collect(java.util.stream.Collectors.toMap(Product::getHoseId, p -> p));

        // 4. Q_required = max(0, Q_net + target_stock - current_stock)
        var result = new HashMap<String, Integer>();
        for (var entry : qNetByHose.entrySet()) {
            String hose = entry.getKey();
            int qNet = entry.getValue();
            Product p = productMap.get(hose);
            if (p == null) {
                throw new IllegalStateException("PRODUCT not found for hose_id: " + hose);
            }
            int qRequired = Math.max(0, qNet + p.getTargetStock() - p.getCurrentStock());
            if (qRequired > 0) {
                result.put(hose, qRequired);
            }
        }
        return result;
    }
}

public record QRequiredResult(
    String hoseId,
    int qNet,                       // 수주 합계
    int targetStock,
    int currentStock,
    int qRequired                   // max(0, qNet + target - current)
) {}
```

## :test_tube: Acceptance Criteria

**검증**: T-U + A

### 4종 재고 케이스 (TC-VC-009)

- [ ] **Case 1 동등**: target=100, current=100, qNet=50 → Q_required = 50 (max(0, 50+100-100))
- [ ] **Case 2 잉여**: target=100, current=200, qNet=50 → Q_required = 0 (max(0, 50+100-200) = -50 → 0)
- [ ] **Case 3 부족**: target=100, current=50, qNet=80 → Q_required = 130 (80+100-50)
- [ ] **Case 4 목표 0**: target=0, current=0, qNet=50 → Q_required = 50

### 추가 케이스

- [ ] **수주 0건** 품번 → result 미포함
- [ ] **호라이즌 외 수주** → 계산 미포함
- [ ] **여러 수주 같은 hose_id** → 합산 (Q_net = sum)
- [ ] **PRODUCT 없는 품번** → IllegalStateException
- [ ] 단위 테스트 ≥ 8 케이스
