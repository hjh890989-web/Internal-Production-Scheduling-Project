#### C. 루트 `build.gradle.kts`

```kotlin
plugins {
    java
    alias(libs.plugins.spring.deps) apply false
    alias(libs.plugins.spring.boot) apply false
}

allprojects {
    group = "com.scheduling"
    version = "0.1.0-SNAPSHOT"

    repositories {
        mavenCentral()
    }
}

subprojects {
    apply(plugin = "java")
    apply(plugin = "io.spring.dependency-management")

    java {
        toolchain {
            languageVersion = JavaLanguageVersion.of(21)
        }
    }

    the<io.spring.gradle.dependencymanagement.dsl.DependencyManagementExtension>().apply {
        imports {
            mavenBom("org.springframework.boot:spring-boot-dependencies:${libs.versions.spring.boot.get()}")
            mavenBom("org.springframework.modulith:spring-modulith-bom:${libs.versions.spring.modulith.get()}")
        }
    }

    dependencies {
        // 모든 subproject에 공통
        "implementation"("org.slf4j:slf4j-api")
        "testImplementation"(libs.bundles.test.core.get())
    }

    tasks.withType<JavaCompile> {
        options.encoding = "UTF-8"
        options.compilerArgs.addAll(listOf("-parameters", "-Xlint:all"))
    }

    tasks.withType<Test> {
        useJUnitPlatform()
        testLogging {
            events("passed", "skipped", "failed")
        }
    }
}
```

#### F. Dockerfile (ST-00-1 backend placeholder 교체)

```dockerfile
# backend/Dockerfile
FROM eclipse-temurin:21-jdk-alpine AS builder
WORKDIR /workspace
COPY . .
RUN ./gradlew :app:bootJar --no-daemon

FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
COPY --from=builder /workspace/app/build/libs/*.jar app.jar
EXPOSE 8080
HEALTHCHECK --interval=10s --timeout=3s --retries=5 --start-period=30s \
    CMD wget --spider --quiet http://localhost:8080/actuator/health/liveness || exit 1
ENTRYPOINT ["java", "-jar", "/app/app.jar"]
```

## :memo: Implementation Notes
- **기술 힌트**:
  - Gradle **Version Catalog**는 멀티모듈에서 의존성 단일 진실 보장 — Maven `dependencyManagement`의 Gradle 버전
  - `useJUnitPlatform()` + `TestLogger` 표준 — Sprint 1 진입 후 testImplementation 추가 시 즉시 사용 가능
  - `eclipse-temurin:21-jdk-alpine` 빌더 + `21-jre-alpine` runtime → 약 200MB (oracle JDK는 400MB+) — NFR-COS-001
  - `forward-headers-strategy: framework` → NGINX `X-Forwarded-Proto` 자동 처리 (TLS termination 뒤에서 HTTPS 인식)
- **유사 패턴**:
  - 다른 Spring Boot 멀티모듈 OSS 참고: Spring Modulith 공식 sample (https://github.com/spring-projects/spring-modulith)
- **표준 참조**:
  - Spring Boot 3.3 Docs: https://docs.spring.io/spring-boot/docs/3.3.x/reference/html/
  - Spring Modulith Docs: https://docs.spring.io/spring-modulith/reference/
  - Gradle Version Catalog: https://docs.gradle.org/current/userguide/platforms.html
