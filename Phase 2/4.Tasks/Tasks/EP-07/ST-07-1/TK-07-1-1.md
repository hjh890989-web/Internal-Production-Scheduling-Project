### 핵심 로직

#### A. `VcConfirmedEvent.java`

```java
package com.scheduling.vc.event;

import java.time.Instant;
import java.util.List;
import java.util.UUID;

/**
 * EP-05 VC 확정 → EP-07/08/09 압출 모듈에 전달.
 * Spring Modulith 모듈 경계 — extrusion 모듈은 vc 도메인 모델 직접 의존 금지,
 * 이벤트 record만 의존.
 */
public record VcConfirmedEvent(
    UUID scheduleId,
    Instant confirmedAt,
    List<VcConfirmedRow> rows
) {
    public record VcConfirmedRow(
        UUID rowId,
        String hoseId,
        java.time.LocalDate productionDate,    // 성형 투입일 — D-1 역산 기준
        String machineId,
        String rotationId,
        int slotNumber,
        int yield                              // 후속 EP-08·EX10 입력
    ) {}
}
```

#### B. `VcConfirmedListener.java`

```java
package com.scheduling.ex.event;

import com.scheduling.vc.event.VcConfirmedEvent;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.modulith.events.ApplicationModuleListener;
import org.springframework.stereotype.Component;

@Component @RequiredArgsConstructor @Slf4j
public class VcConfirmedListener {

    private final ExtrusionScheduleService extrusionService;

    /**
     * Spring Modulith @ApplicationModuleListener — 별도 트랜잭션 + async.
     * VC 트랜잭션 commit 후 처리, 실패해도 VC commit은 유지.
     */
    @ApplicationModuleListener
    public void onVcConfirmed(VcConfirmedEvent event) {
        log.info("VC confirmed event received: scheduleId={}, rows={}",
            event.scheduleId(), event.rows().size());

        try {
            extrusionService.generateCandidates(event);
        } catch (Exception e) {
            log.error("압출 후보 생성 실패: scheduleId={}", event.scheduleId(), e);
            throw e;       // Spring Modulith가 retry/DLQ로 처리
        }
    }
}
```

#### C. `VcConfirmedPublisher.java` (EP-05 측 수정)

```java
@Service @RequiredArgsConstructor
public class VcConfirmedPublisher {

    private final ApplicationEventPublisher publisher;
    private final VcScheduleRepository scheduleRepo;

    @Transactional
    public void publishConfirmed(UUID scheduleId) {
        var rows = scheduleRepo.findRowsByScheduleId(scheduleId).stream()
            .map(r -> new VcConfirmedEvent.VcConfirmedRow(
                r.getId(), r.getHoseId(), r.getProductionDate(),
                r.getMachineId(), r.getRotationId().name(),
                r.getSlotNumber(), r.getYield()))
            .toList();

        publisher.publishEvent(new VcConfirmedEvent(scheduleId, Instant.now(), rows));
    }
}
```
