#### A. `OrderType.java` — ordinal 순서가 우선순위

```java
package com.scheduling.order.domain;

/**
 * BR-O01: ordinal 작을수록 약한 우선순위
 * - FORECAST(0)  : 월별 예상 — 가장 약함
 * - KD(1)        : KD 발주
 * - WEEKLY(2)    : 주간 발주
 * - CONFIRMED(3) : 확정 — 가장 강함
 *
 * 정책 변경 시 enum 정의 순서만 수정 (코드 if-else 무관)
 */
public enum OrderType {
    FORECAST("FORECAST", "월별 예상"),
    KD("KD", "KD 발주"),
    WEEKLY("WEEKLY", "주간 발주"),
    CONFIRMED("CONFIRMED", "확정");

    private final String code;
    private final String displayName;

    OrderType(String code, String displayName) {
        this.code = code;
        this.displayName = displayName;
    }

    public int precedenceRank() { return ordinal(); }
    public boolean isStrongerThan(OrderType other) { return this.ordinal() > other.ordinal(); }
}
```

import java.util.List;

public record Resolution(
    OrderKey key,
    OrderDraft winner,             // 정본
    List<OrderDraft> losers,       // 패자
    Order existingMaster,          // 기존 마스터 (있으면)

}

#### C. `PrecedenceResolver.java` — 핵심

```java
package com.scheduling.order.domain;

import org.springframework.stereotype.Component;

import java.util.*;
import java.util.stream.Stream;

@Component
public class PrecedenceResolver {

    /**
     * DuplicateGroup → Resolution.
     * BR-O01: ordinal max가 winner.
     */
    public Resolution resolve(DuplicateGroup group) {
        // 1. 모든 후보(candidates + existing) 통합 후 ordinal 비교
        Stream<OrderDraft> stream = group.candidates().stream();
        OrderDraft existingAsDraft = group.hasExisting()
            ? OrderDraft.fromExisting(group.existingMaster())
            : null;

        OrderDraft winner = stream
            .reduce((a, b) -> a.orderType().isStrongerThan(b.orderType()) ? a : b)
            .orElseThrow();

        if (existingAsDraft != null && existingAsDraft.orderType().isStrongerThan(winner.orderType())) {
            // 기존 마스터가 더 강함 → 유지
            return new Resolution(
                group.key(),
                existingAsDraft,
                group.candidates(),    // 모든 신규가 패자
                group.existingMaster(),
                Resolution.Decision.KEPT_EXISTING
            );
        }

        List<OrderDraft> losers = group.candidates().stream()
            .filter(d -> d != winner)
            .toList();

        Resolution.Decision decision = group.hasExisting()
            ? Resolution.Decision.REPLACED_EXISTING
            : Resolution.Decision.NEW_WINS;

        return new Resolution(group.key(), winner, losers, group.existingMaster(), decision);
    }
}
```

#### D. `ResolutionService` — 통합

```java
@Service
@RequiredArgsConstructor
public class ResolutionService {

    private final DuplicateDetectionService detector;
    private final PrecedenceResolver resolver;
    private final PrecedenceAuditLogger auditLogger;   // TK-02-2-2

    public List<Resolution> resolveAll(List<OrderDraft> batch) {
        var groups = detector.detect(batch);
        return groups.stream()
            .map(g -> {
                Resolution r = resolver.resolve(g);
                auditLogger.log(r);   // BR-X02 자동 기록
                return r;
            })
            .toList();
    }
}
```
