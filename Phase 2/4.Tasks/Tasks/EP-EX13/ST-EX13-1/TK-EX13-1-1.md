### `VcChangedEvent.java`

```java
public record VcChangedEvent(
    UUID scheduleId,
    Instant changedAt,
    String changedBy,
    ChangeType changeType,                       // QUANTITY · DATE · HOSE · OVERRIDE
    List<VcChangedRow> rows
) {
    public record VcChangedRow(
        UUID rowId,
        String hoseId,
        LocalDate oldProductionDate,
        LocalDate newProductionDate,
        int oldYield,
        int newYield
    ) {}

    public enum ChangeType {
        QUANTITY,      // yield 변경
        DATE,          // production_date 변경
        HOSE,          // hose_id 교체 (override)
        OVERRIDE       // EP-13 일중 락 override
    }
}
```

### `VcChangedPublisher.java`

```java
@Service @RequiredArgsConstructor
public class VcChangedPublisher {

    private final ApplicationEventPublisher publisher;

    @Transactional
    public void publishChanged(UUID scheduleId, ChangeType type, List<VcChangedRow> rows) {
        publisher.publishEvent(new VcChangedEvent(
            scheduleId, Instant.now(),
            SecurityContextHolder.getContext().getAuthentication().getName(),
            type, rows
        ));
    }
}
```

### `VcChangedListener.java`

```java
@Component @RequiredArgsConstructor @Slf4j
public class VcChangedListener {

    private final ExScheduleReplanService replanService;     // TK-EX13-1-3

    @ApplicationModuleListener
    public void onVcChanged(VcChangedEvent event) {
        log.info("VC changed: scheduleId={}, type={}, rows={}",
            event.scheduleId(), event.changeType(), event.rows().size());

        replanService.partialReplan(event);
    }
}
```
