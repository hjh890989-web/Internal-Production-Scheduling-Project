### `ScheduleValidatorService.java`

```java
@Service @RequiredArgsConstructor
public class ScheduleValidatorService {

    private final VcScheduleRepository scheduleRepo;
    private final RulePipeline pipeline;
    private final ConflictCategorizer categorizer;

    /**
     * REQ-FUNC-VC-016: 확정 스케줄 전체 row를 RulePipeline으로 점검.
     * - 모든 위반 수집 (early-exit 금지)
     * - parallel stream — row 단위 독립 검증
     * - ledger는 row 단위 재구성 (현재 상태 reflect)
     */
    public ValidationResult validateAll(UUID scheduleId) {
        var rows = scheduleRepo.findByScheduleId(scheduleId);
        var ledger = AllocationLedger.fromConfirmed(rows);     // 기존 점유 반영

        var violations = rows.parallelStream()
            .map(row -> {
                var cand = SlotCandidate.fromConfirmed(row);
                // 본 row를 ledger에서 제외하고 재평가 (자기 자신과 중복 회피)
                var localLedger = ledger.excluding(row);
                var result = pipeline.evaluate(cand, localLedger);
                if (result.passed()) return null;
                var conflict = AllocationConflict.fromRule(row, result);
                var classified = categorizer.classify(conflict);
                return new ValidationViolation(row.id(), row.hoseId(),
                    classified.category(), result.message());
            })
            .filter(Objects::nonNull)
            .toList();

        return new ValidationResult(scheduleId, violations, summarize(violations), Instant.now());
    }

    private Map<ConflictCategory, Long> summarize(List<ValidationViolation> v) {
        return v.stream().collect(Collectors.groupingBy(
            ValidationViolation::category, Collectors.counting()));
    }
}

public record ValidationResult(UUID scheduleId, List<ValidationViolation> violations,
                                Map<ConflictCategory, Long> summary, Instant executedAt) {}

public record ValidationViolation(UUID rowId, String hoseId,
                                   ConflictCategory category, String reason) {}
```

### `ScheduleValidateController.java`

```java
@RestController @RequestMapping("/api/v1/schedule/validate-all") @RequiredArgsConstructor
public class ScheduleValidateController {

    private final ScheduleValidatorService validator;

    @PostMapping("/{scheduleId}")
    @PreAuthorize("isAuthenticated()")
    public ValidationResult validate(@PathVariable UUID scheduleId) {
        return validator.validateAll(scheduleId);
    }
}
```
