#### B. `SeverityClassifier.java`

```java
@Component @RequiredArgsConstructor
public class SeverityClassifier {

    private final SeverityConfig config;

    public Severity classify(RowDiff diff) {
        // BR-O02: NEW·DELETED는 자동 Critical (수주 변동의 기본 중요성)
        if (diff.type() == DiffType.NEW && config.isNewAlwaysCritical()) return Severity.CRITICAL;
        if (diff.type() == DiffType.DELETED && config.isDeletedAlwaysCritical()) return Severity.CRITICAL;
        if (diff.type() == DiffType.UNCHANGED) return Severity.NORMAL;

        // MODIFIED — 필드별 평가
        for (FieldDiff fd : diff.fieldDiffs()) {
            switch (fd.fieldName()) {
                case "delivery_date":
                    if (config.isDeliveryDateAlwaysCritical()) return Severity.CRITICAL;
                    break;
                case "hose_id":
                    if (config.isHoseIdAlwaysCritical()) return Severity.CRITICAL;
                    break;
                case "qty":
                    if (qtyChangeOverThreshold(fd)) return Severity.CRITICAL;
                    break;
                // customer·order_type 변경 → Normal (수량·납기 외)
            }
        }
        return Severity.NORMAL;
    }

    private boolean qtyChangeOverThreshold(FieldDiff fd) {
        if (!(fd.before() instanceof Number before) || !(fd.after() instanceof Number after)) {
            return true;                                  // 알 수 없으면 conservative
        }
        double oldVal = before.doubleValue();
        double newVal = after.doubleValue();
        if (oldVal == 0) return newVal != 0;              // 0 → N 도 Critical
        double pct = Math.abs(newVal - oldVal) / oldVal;
        return pct >= config.getQtyChangeThresholdPct();
    }
}
```

#### C. Service 통합 (DiffResult 후처리)

```java
@Service @RequiredArgsConstructor
public class DiffTaggingService {

    private final DiffEngineService diffEngine;
    private final SeverityClassifier classifier;
    private final OrderChangeRepository repo;

    @Transactional
    public DiffResult computeAndTag(UUID trackingId, List<OrderDraft> batch, int prevVersion) {
        DiffResult diff = diffEngine.compute(trackingId, batch, prevVersion);

        // 각 RowDiff에 severity 부여 후 DB severity 컬럼 UPDATE
        diff.rows().stream()
            .filter(r -> r.type() != DiffType.UNCHANGED)
            .forEach(r -> {
                Severity sev = classifier.classify(r);
                repo.updateSeverityByTrackingAndKey(trackingId, r.key(), sev.name());
            });

        return diff;
    }
}
```
