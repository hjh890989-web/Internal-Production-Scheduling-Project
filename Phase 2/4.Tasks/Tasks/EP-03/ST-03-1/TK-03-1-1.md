#### A. Enum + DTO

```java
public enum DiffType { NEW, MODIFIED, DELETED, UNCHANGED }

public record FieldDiff(
    String fieldName,         // "delivery_date" · "qty" · "hose_id" · "customer"
    Object before,
    Object after
) {}

public record RowDiff(
    OrderKey key,
    DiffType type,
    OrderDraft newRow,         // NEW·MODIFIED·UNCHANGED 시 not null
    Order oldRow,              // DELETED·MODIFIED·UNCHANGED 시 not null
    List<FieldDiff> fieldDiffs // MODIFIED만 의미 있음
) {}

public record DiffResult(
    UUID trackingId,
    int previousVersion,
    int newVersion,
    Instant computedAt,
    List<RowDiff> rows
) {
    public int countByType(DiffType t) {
        return (int) rows.stream().filter(r -> r.type() == t).count();
    }
}
```

#### B. `DiffEngineService.java` — 핵심

```java
@Service @RequiredArgsConstructor
public class DiffEngineService {

    private final OrderRepository repo;

    /**
     * 신규 (resolved winners) vs 이전 마스터 outer join.
     */
    public DiffResult compute(UUID trackingId, List<OrderDraft> newRows, int previousVersion) {
        // 1. 이전 마스터 한 번에 조회
        List<Order> oldRows = repo.findByMasterVersion(previousVersion);

        // 2. 좌·우 Map (OrderKey 기준)
        Map<OrderKey, Order> oldByKey = oldRows.stream()
            .collect(Collectors.toMap(OrderKey::ofEntity, o -> o));
        Map<OrderKey, OrderDraft> newByKey = newRows.stream()
            .collect(Collectors.toMap(OrderKey::ofDraft, d -> d));

        // 3. 합집합 키
        Set<OrderKey> allKeys = new HashSet<>();
        allKeys.addAll(oldByKey.keySet());
        allKeys.addAll(newByKey.keySet());

        // 4. 키별 분류
        List<RowDiff> rowDiffs = allKeys.stream()
            .map(key -> classify(key, oldByKey.get(key), newByKey.get(key)))
            .toList();

        return new DiffResult(trackingId, previousVersion, previousVersion + 1,
            Instant.now(), rowDiffs);
    }

    private RowDiff classify(OrderKey key, Order old, OrderDraft fresh) {
        if (old == null && fresh != null) {
            return new RowDiff(key, DiffType.NEW, fresh, null, List.of());
        }
        if (old != null && fresh == null) {
            return new RowDiff(key, DiffType.DELETED, null, old, List.of());
        }
        // 양쪽 존재 — 필드 비교
        List<FieldDiff> diffs = compareFields(old, fresh);
        DiffType type = diffs.isEmpty() ? DiffType.UNCHANGED : DiffType.MODIFIED;
        return new RowDiff(key, type, fresh, old, diffs);
    }

    private List<FieldDiff> compareFields(Order old, OrderDraft fresh) {
        var diffs = new ArrayList<FieldDiff>();
        compare("hose_id", old.getHoseId(), fresh.hoseId(), diffs);
        compare("delivery_date", old.getDeliveryDate(), fresh.deliveryDate(), diffs);
        compare("qty", old.getQty(), fresh.qty(), diffs);
        compare("customer", old.getCustomer(), fresh.customer(), diffs);
        compare("order_type", old.getOrderType(), fresh.orderType(), diffs);
        return diffs;
    }

    private void compare(String field, Object before, Object after, List<FieldDiff> diffs) {
        if (!Objects.equals(before, after)) {
            diffs.add(new FieldDiff(field, before, after));
        }
    }
}
```

## :test_tube: Acceptance Criteria

**검증**: T-U + A

- [ ] **NEW 분류**: 빈 마스터 + 신규 100 row → 100 RowDiff.NEW
- [ ] **DELETED 분류**: 마스터 50 row + 빈 신규 → 50 RowDiff.DELETED
- [ ] **MODIFIED 분류**: 같은 키 + 필드 1개 변경 → MODIFIED + fieldDiffs.size()==1
- [ ] **다중 필드 MODIFIED**: 같은 키 + 필드 3개 변경 → MODIFIED + fieldDiffs.size()==3
- [ ] **UNCHANGED 분류**: 같은 키 + 모든 필드 동일 → UNCHANGED + fieldDiffs.empty
- [ ] **혼합 시나리오**: 100 row 중 30 NEW + 20 DELETED + 25 MODIFIED + 25 UNCHANGED → 정확 분류
- [ ] **성능**: 10,000 vs 10,000 row diff p95 ≤ 5초
- [ ] **null 안전**: customer null이거나 빈 문자열이면 정규화 후 비교 ("내수" 기본값)
