### `ReasonValidator.java`

```java
@Component
public class ReasonValidator {

    private static final int MIN_LENGTH = 30;
    private static final Set<String> PLACEHOLDER_BLACKLIST = Set.of(
        "test", "임시", "...", "...", "asdf", "qwer", "ㅁㄴㅇㄹ", "ㅇㅇ", "ㄱㄱ"
    );

    /**
     * REQ-FUNC-CO-010: override 사유 검증.
     *  - 30자 이상 (공백 제외)
     *  - placeholder/의미 없는 입력 차단
     *  - 위반 시 ReasonValidationException
     */
    public void requireValid(String reason) {
        if (reason == null || reason.isBlank()) {
            throw new ReasonValidationException("override 사유는 필수입니다");
        }

        String trimmed = reason.strip();
        String stripped = trimmed.replaceAll("\\s+", "");

        if (stripped.length() < MIN_LENGTH) {
            throw new ReasonValidationException(
                "override 사유는 공백 제외 %d자 이상이어야 합니다 (현재: %d자)"
                    .formatted(MIN_LENGTH, stripped.length()));
        }

        var lower = trimmed.toLowerCase(Locale.ROOT);
        for (var ph : PLACEHOLDER_BLACKLIST) {
            if (lower.equals(ph) || lower.startsWith(ph + " ")) {
                throw new ReasonValidationException(
                    "의미 있는 사유를 입력해 주세요 (placeholder 차단)");
            }
        }
    }
}

public class ReasonValidationException extends RuntimeException {
    public ReasonValidationException(String msg) { super(msg); }
}
```

### `GlobalExceptionHandler.java` 추가

```java
@ExceptionHandler(ReasonValidationException.class)
public ResponseEntity<ErrorResponse> handleReasonValidation(ReasonValidationException e) {
    var body = ErrorResponse.builder()
        .code("REASON_VALIDATION")
        .message(e.getMessage())
        .build();
    return ResponseEntity.status(HttpStatus.BAD_REQUEST).body(body);
}
```
