### `VcLockOverrideService.java`

```java
@Service @RequiredArgsConstructor
public class VcLockOverrideService {

    private final VcScheduleRepository repo;
    private final ReasonValidator reasonValidator;          // TK-13-4-2

    @Auditable(reason = "'BR-V07 override: ' + #request.reason()")
    @Transactional
    public VcSchedule override(UUID scheduleId, OverrideRequest request) {
        reasonValidator.requireValid(request.reason());

        var existing = repo.findById(scheduleId)
            .orElseThrow(() -> new NotFoundException("schedule=" + scheduleId));

        // 같은 슬롯의 기존 row delete + new hose insert (DEFERRED constraint)
        repo.deleteByMachineAndSlotAndDate(
            existing.getMachineId(),
            existing.getSlotPosition(),
            existing.getProductionDate());

        var newSchedule = existing.toBuilder()
            .id(UUID.randomUUID())
            .hoseId(request.newHoseId())
            .build();

        return repo.save(newSchedule);
        // transaction commit 시점에 UNIQUE constraint 재검증 (1 row만 남음 → 통과)
    }
}
```

### `VcLockOverrideController.java`

```java
@RestController @RequestMapping("/api/v1/schedule/vc") @RequiredArgsConstructor
public class VcLockOverrideController {

    private final VcLockOverrideService service;

    @PostMapping("/{id}/override-intraday-lock")
    @PreAuthorize("hasRole('PLANNER')")
    public ResponseEntity<VcSchedule> override(
        @PathVariable UUID id,
        @Valid @RequestBody OverrideRequest request
    ) {
        var updated = service.override(id, request);
        return ResponseEntity.ok(updated);
    }
}
```
