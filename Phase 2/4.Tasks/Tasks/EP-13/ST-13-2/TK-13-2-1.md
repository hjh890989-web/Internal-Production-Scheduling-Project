### `IntraDayLockRule.java`

```java
@Component @RequiredArgsConstructor
public class IntraDayLockRule {

    /**
     * BR-V07: 같은 (machine_id, slot_position, production_date) 슬롯에 다른 hose 배치 차단.
     *  - ledger에 동일 키 hose 없음 → pass
     *  - ledger에 같은 hose 있음 → pass (중복 배치, slot count rule이 별도 처리)
     *  - ledger에 다른 hose 있음 → fail
     */
    public RuleResult evaluate(SlotCandidate cand, AllocationLedger ledger) {
        var existingHose = ledger.hoseAt(cand.machineId(), cand.slotNumber(), cand.productionDate());

        if (existingHose.isEmpty()) return RuleResult.pass();
        if (existingHose.get().equals(cand.hoseId())) return RuleResult.pass();

        return RuleResult.fail("BR-V07 일중 교체 차단: %s/%d/%s에 %s 이미 배치 (%s 시도)"
            .formatted(cand.machineId(), cand.slotNumber(), cand.productionDate(),
                        existingHose.get(), cand.hoseId()));
    }
}
```

### `AllocationLedger.java` 수정

```java
public class AllocationLedger {
    // ... 기존 필드

    // (machine, slot, date) → hose_id (BR-V07 검증용)
    private final Map<SlotKey, String> slotHoseMap = new HashMap<>();

    public record SlotKey(String machineId, int slot, LocalDate date) {}

    public Optional<String> hoseAt(String machineId, int slot, LocalDate date) {
        return Optional.ofNullable(slotHoseMap.get(new SlotKey(machineId, slot, date)));
    }

    public void assignHose(String machineId, int slot, LocalDate date, String hoseId) {
        slotHoseMap.put(new SlotKey(machineId, slot, date), hoseId);
    }

    /**
     * Allocator가 row 배치 확정 시 호출.
     */
    public void recordAssignment(SlotCandidate cand) {
        assignHose(cand.machineId(), cand.slotNumber(), cand.productionDate(), cand.hoseId());
        // 기존 slot count·angle count도 함께 갱신
        addSlots(cand.hoseId(), cand.productionDate(), cand.machineId(), cand.rotationId(), 1);
    }
}
```

### `RulePipeline.java` 통합

```java
@Component @RequiredArgsConstructor
public class RulePipeline {

    private final SlotOxRule slotOxRule;
    private final MachinePinRule machinePinRule;
    private final LeftRightRule leftRightRule;
    private final HoseSlotCapRule hoseSlotCapRule;
    private final SpecLt7CapRule specLt7CapRule;
    private final IntraDayLockRule intraDayLockRule;     // ← 추가

    public RuleResult evaluate(SlotCandidate cand, AllocationLedger ledger) {
        var r1 = slotOxRule.evaluate(cand);              if (!r1.passed()) return r1;
        var r2 = machinePinRule.evaluate(cand);          if (!r2.passed()) return r2;
        var r3 = leftRightRule.evaluate(cand);           if (!r3.passed()) return r3;
        var r4 = hoseSlotCapRule.evaluate(cand, ledger); if (!r4.passed()) return r4;
        var r5 = specLt7CapRule.evaluate(cand, ledger);  if (!r5.passed()) return r5;
        return intraDayLockRule.evaluate(cand, ledger);  // ← 마지막
    }
}
```
