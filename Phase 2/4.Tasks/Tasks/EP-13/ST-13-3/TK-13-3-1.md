### `BusinessDayBoundaryFormatter.java`

```java
@Component @RequiredArgsConstructor
public class BusinessDayBoundaryFormatter {

    private final WorkingCalendarService calendar;
    private static final DateTimeFormatter ISO = DateTimeFormatter.ofPattern("yyyy-MM-dd");

    /**
     * 해당 영업일의 종료 경계 키.
     *  - 평일: "2026-03-02_END" (3/2 23:59:59까지, 3/3 00:00 다음 영업일)
     *  - 연휴 직전: "2026-02-13_END" → 다음 영업일 2/19 (설날 휴일 건너뜀)
     */
    public String formatBoundary(LocalDate date) {
        return ISO.format(date) + "_END";
    }

    /**
     * 다음 영업일 (락 해제 시점).
     */
    public LocalDate nextBusinessDay(LocalDate date) {
        return calendar.addWorkingDays(date, 1);
    }

    /**
     * "락 유지 기간" 사람 친화 문자열.
     * 예: "2026-03-02 ~ 2026-03-03 시작 전" (정상)
     *     "2026-02-13 ~ 2026-02-19 시작 전" (연휴 후 해제)
     */
    public String formatLockPeriod(LocalDate productionDate) {
        LocalDate next = nextBusinessDay(productionDate);
        return "%s ~ %s 시작 전".formatted(ISO.format(productionDate), ISO.format(next));
    }
}
```

### `Do04Renderer.java` 수정

```java
@Component @RequiredArgsConstructor
public class Do04Renderer {

    private final BusinessDayBoundaryFormatter boundaryFormatter;

    public Do04Document render(LocalDate date, List<VcSchedule> rows) {
        var doc = new Do04Document();
        doc.setProductionDate(date);
        doc.setBoundaryKey(boundaryFormatter.formatBoundary(date));      // ← 추가
        doc.setLockPeriod(boundaryFormatter.formatLockPeriod(date));     // ← 추가
        doc.setRows(rows);
        return doc;
    }
}
```
