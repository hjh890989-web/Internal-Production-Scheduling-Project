### `IntraDayConflictException.java`

```java
package com.scheduling.vc.lock;

import java.time.LocalDate;

public class IntraDayConflictException extends RuntimeException {

    private final String machineId;
    private final int slotPosition;
    private final LocalDate productionDate;
    private final String hoseId;

    public IntraDayConflictException(String machineId, int slotPosition,
                                       LocalDate productionDate, String hoseId, Throwable cause) {
        super("BR-V07 당일 락 위반: %s 슬롯 %d %s에 %s 배치 시도 — 동일 슬롯에 다른 호스 존재"
            .formatted(machineId, slotPosition, productionDate, hoseId), cause);
        this.machineId = machineId;
        this.slotPosition = slotPosition;
        this.productionDate = productionDate;
        this.hoseId = hoseId;
    }

    // getters...
}
```

### `IntraDayConstraintViolationTranslator.java`

```java
@Component @RequiredArgsConstructor @Slf4j
public class IntraDayConstraintViolationTranslator {

    private static final String SQL_STATE_UNIQUE_VIOLATION = "23505";
    private static final String CONSTRAINT_NAME = "uk_vc_schedule_intraday_lock";

    /**
     * PSQLException 분석 — IntraDayConflictException 또는 원본 그대로 rethrow.
     */
    public RuntimeException translate(DataIntegrityViolationException e, VcSchedule attempted) {
        Throwable cause = e.getCause();
        if (!(cause instanceof PSQLException psql)) return e;

        var serverError = psql.getServerErrorMessage();
        if (serverError == null) return e;

        if (SQL_STATE_UNIQUE_VIOLATION.equals(psql.getSQLState())
            && CONSTRAINT_NAME.equals(serverError.getConstraint())) {

            log.warn("BR-V07 당일 락 violation: hose={} machine={} slot={} date={}",
                attempted.getHoseId(), attempted.getMachineId(),
                attempted.getSlotPosition(), attempted.getProductionDate());

            return new IntraDayConflictException(
                attempted.getMachineId(),
                attempted.getSlotPosition(),
                attempted.getProductionDate(),
                attempted.getHoseId(),
                psql
            );
        }

        return e;
    }
}
```

### `GlobalExceptionHandler.java` 수정

```java
@ControllerAdvice @Slf4j
public class GlobalExceptionHandler {

    @ExceptionHandler(IntraDayConflictException.class)
    public ResponseEntity<ErrorResponse> handleIntraDayConflict(IntraDayConflictException e) {
        var body = ErrorResponse.builder()
            .code("BR_V07_INTRADAY_LOCK")
            .message(e.getMessage())
            .details(Map.of(
                "machineId", e.getMachineId(),
                "slotPosition", e.getSlotPosition(),
                "productionDate", e.getProductionDate(),
                "attemptedHoseId", e.getHoseId()
            ))
            .build();

        return ResponseEntity.status(HttpStatus.CONFLICT).body(body);
    }
}
```

### Service에서 사용

```java
@Service @RequiredArgsConstructor
public class VcScheduleService {

    private final VcScheduleRepository repo;
    private final IntraDayConstraintViolationTranslator translator;

    public VcSchedule save(VcSchedule schedule) {
        try {
            return repo.save(schedule);
        } catch (DataIntegrityViolationException e) {
            throw translator.translate(e, schedule);
        }
    }
}
```
