### `MetersPerMinute.java`

```java
package com.scheduling.ex.yield.unit;

import java.math.BigDecimal;

/**
 * 압출 speed — m/min 단위 type-safe wrapper.
 *
 * 마스터 컬럼명: `speed_m_per_min`. mm/min 또는 m/sec 입력 차단.
 * 운영자가 명시적 단위 표시한 값만 수용.
 */
public record MetersPerMinute(BigDecimal value) {

    public MetersPerMinute {
        if (value == null) throw new IllegalArgumentException("speed value is null");
        if (value.signum() <= 0)
            throw new UnitMismatchException("speed must be > 0 m/min, got: " + value);
        // 정상 범위: 1.0 ~ 50.0 m/min (현장 실측)
        if (value.compareTo(BigDecimal.valueOf(100)) > 0)
            throw new UnitMismatchException(
                "speed %s suspiciously high — possibly mm/min? expected m/min".formatted(value));
    }

    public static MetersPerMinute of(double value) {
        return new MetersPerMinute(BigDecimal.valueOf(value));
    }

    public BigDecimal asMillimetersPerMinute() {
        return value.multiply(BigDecimal.valueOf(1000));
    }
}
```

### `Millimeters.java`

```java
package com.scheduling.ex.yield.unit;

/**
 * 호스 length — mm 단위 type-safe wrapper.
 *
 * 마스터 컬럼명: `length_mm`. m 또는 cm 입력 차단.
 */
public record Millimeters(int value) {

    public Millimeters {
        if (value <= 0)
            throw new UnitMismatchException("length must be > 0 mm, got: " + value);
        // 정상 범위: 100 ~ 10000 mm (현장 실측 — 호스 길이 0.1m ~ 10m)
        if (value < 10)
            throw new UnitMismatchException(
                "length %d mm suspiciously low — possibly meters? expected mm".formatted(value));
        if (value > 100_000)
            throw new UnitMismatchException(
                "length %d mm suspiciously high — possibly micrometers? expected mm".formatted(value));
    }
}
```

### `MetersPerMinuteConverter.java`

```java
@Converter(autoApply = true)
public class MetersPerMinuteConverter implements AttributeConverter<MetersPerMinute, BigDecimal> {

    @Override
    public BigDecimal convertToDatabaseColumn(MetersPerMinute attr) {
        return attr == null ? null : attr.value();
    }

    @Override
    public MetersPerMinute convertToEntityAttribute(BigDecimal dbData) {
        return dbData == null ? null : new MetersPerMinute(dbData);
    }
}
```

### `YieldFormula.java` 수정 — wrapper 입력

```java
@Component
public class YieldFormula {

    public long compute(MetersPerMinute speed, int effectiveMin, Millimeters length) {
        if (effectiveMin <= 0)
            throw new IllegalArgumentException("effective_min must be > 0");

        BigDecimal totalMm = speed.value()
            .multiply(BigDecimal.valueOf(effectiveMin))
            .multiply(BigDecimal.valueOf(1000));

        BigDecimal yieldExact = totalMm.divide(
            BigDecimal.valueOf(length.value()), 10, RoundingMode.HALF_UP);

        return yieldExact.setScale(0, RoundingMode.FLOOR).longValueExact();
    }

    // 기존 BigDecimal/int signature deprecated — 마이그레이션 기간
    @Deprecated
    public long compute(BigDecimal speed, int min, int lengthMm) {
        return compute(new MetersPerMinute(speed), min, new Millimeters(lengthMm));
    }
}
```
