### `ShiftCache.java`

```java
@Component @RequiredArgsConstructor
public class ShiftCache {

    private final ShiftRepository repo;
    private final Cache<String, Shift> cache = Caffeine.newBuilder()
        .maximumSize(20)                      // 4 shift이지만 향후 확장 여유
        .expireAfterWrite(Duration.ofHours(1))
        .recordStats()
        .build();

    public Optional<Shift> get(String shiftCode) {
        var cached = cache.getIfPresent(shiftCode);
        if (cached != null) return Optional.of(cached);
        return repo.findById(shiftCode).map(s -> { cache.put(shiftCode, s); return s; });
    }

    public List<Shift> all() {
        return repo.findAllByOrderBySortOrderAsc();
    }

    public void invalidate(String shiftCode) { cache.invalidate(shiftCode); }
    public void invalidateAll() { cache.invalidateAll(); }
}
```

### `ShiftEfficiencyService.java`

```java
@Service @RequiredArgsConstructor
public class ShiftEfficiencyService {

    private final ShiftCache cache;

    /**
     * BR-E04: shift_code → effective_min.
     * yield 수식 입력원.
     */
    public int effectiveMin(String shiftCode) {
        return cache.get(shiftCode)
            .map(Shift::getEffectiveMin)
            .orElseThrow(() -> new MasterNotFoundException("shift=" + shiftCode));
    }

    public BigDecimal efficiency(String shiftCode) {
        return cache.get(shiftCode)
            .map(Shift::getEfficiency)
            .orElseThrow(() -> new MasterNotFoundException("shift=" + shiftCode));
    }

    public int nominalMin(String shiftCode) {
        return cache.get(shiftCode)
            .map(Shift::getNominalMin)
            .orElseThrow(() -> new MasterNotFoundException("shift=" + shiftCode));
    }

    /**
     * 4 shift × effective_min 일괄 조회 (그룹핑·검증 게이트용).
     */
    public Map<String, Integer> effectiveMinAllShifts() {
        return cache.all().stream()
            .collect(Collectors.toMap(Shift::getShiftCode, Shift::getEffectiveMin));
    }
}
```
