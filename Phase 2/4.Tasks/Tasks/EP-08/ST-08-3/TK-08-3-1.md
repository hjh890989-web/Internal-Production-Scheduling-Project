### `ExtrusionDemandCalculator.java`

```java
@Service @RequiredArgsConstructor
public class ExtrusionDemandCalculator {

    private final ProductInventoryCache inventoryCache;

    /**
     * REQ-FUNC-EX-010: Q_ext = max(0, Q_vc + target - current)
     *  - 재고 충분 (current ≥ target + Q_vc + ε) → 0
     *  - 재고 부족 → Q_vc + target - current
     */
    public long computeQExt(long qVc, int targetStock, int currentStock) {
        if (qVc < 0) throw new IllegalArgumentException("qVc must be ≥ 0: " + qVc);
        if (targetStock < 0) throw new IllegalArgumentException("target must be ≥ 0: " + targetStock);
        if (currentStock < 0) throw new IllegalArgumentException("current must be ≥ 0: " + currentStock);

        long need = qVc + targetStock - currentStock;
        return Math.max(0L, need);
    }

    /**
     * 마스터 자동 조회 — 재고 누락 품번은 target=0, current=0 가정 (Q_ext = Q_vc).
     */
    public long computeQExtForHose(String hoseId, long qVc) {
        var inventory = inventoryCache.get(hoseId).orElse(
            ProductInventory.empty(hoseId));
        return computeQExt(qVc, inventory.getTargetStock(), inventory.getCurrentStock());
    }
}
```

    @Id @Column(name="hose_id") private String hoseId;
    @Column(name="target_stock", nullable=false) private Integer targetStock;
    @Column(name="current_stock", nullable=false) private Integer currentStock;
    @Column(name="last_counted_at") private Instant lastCountedAt;

    public static ProductInventory empty(String hoseId) {
        return ProductInventory.builder()
            .hoseId(hoseId).targetStock(0).currentStock(0).build();
    }
}

- [ ] **`computeQExt(100, 50, 30)` = 120** (100 + 50 − 30)
- [ ] **`computeQExt(100, 50, 200)` = 0** (current ≥ target + qVc)

- [ ] **`computeQExt(100, -1, 30)` → IllegalArgumentException**
