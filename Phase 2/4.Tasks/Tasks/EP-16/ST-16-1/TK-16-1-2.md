### `FallbackScheduler.java`

```java
@Component @RequiredArgsConstructor @Slf4j
public class FallbackScheduler {

    private static final Duration READ_DEADLINE = Duration.ofMinutes(1);

    private final DeliveryLogRepository logRepo;
    private final KakaoBizMessageService kakaoService;
    private final NotificationRepository notificationRepo;

    /**
     * 매 30초 — 이메일 발송 후 1분 미READ 알림 검출 → 카톡 fallback.
     */
    @Scheduled(fixedDelay = 30_000)
    public void runFallbackCheck() {
        Instant cutoff = Instant.now().minus(READ_DEADLINE);

        var pendingFallbacks = logRepo.findEmailUnreadAfter(cutoff);

        for (var log : pendingFallbacks) {
            var notification = notificationRepo.findById(log.getNotificationId()).orElse(null);
            if (notification == null) continue;

            // Critical 또는 HIGH 우선순위만 fallback
            if (notification.getPriority() != NotificationPriority.HIGH) continue;

            // 이미 카톡 발송 이력 있는지 확인 (중복 방지)
            if (logRepo.existsKakaoForNotification(notification.getId())) continue;

            log.info("Fallback 트리거: notification={}, 이메일 미READ", notification.getId());

            try {
                kakaoService.send(notification.getId(),
                    notification.getRecipient(),
                    notification.getMessage());
            } catch (Exception e) {
                log.error("Kakao fallback 실패: notification={}", notification.getId(), e);
            }
        }
    }
}
```

### `NotificationPriority.java`

```java
public enum NotificationPriority {
    HIGH,    // 납기 위반·당일 락 위반 등 — fallback 활성
    MEDIUM,  // 정상 운영 알림 — 이메일만
    LOW      // 정보성 — 이메일만
}
```

### `DeliveryLogRepository.java` (쿼리 추가)

```java
@Query("""
    SELECT log FROM DeliveryLog log
    WHERE log.channel = 'EMAIL'
      AND log.status IN ('SENT', 'DELIVERED')
      AND log.readAt IS NULL
      AND log.sentAt <= :cutoff
""")
List<DeliveryLog> findEmailUnreadAfter(@Param("cutoff") Instant cutoff);

@Query("""
    SELECT COUNT(*) > 0 FROM DeliveryLog log
    WHERE log.notificationId = :notificationId
      AND log.channel = 'KAKAO'
""")
boolean existsKakaoForNotification(@Param("notificationId") UUID notificationId);
```
