#### C. `SourceClassifierService.java`

```java
package com.scheduling.order.parser;

import lombok.RequiredArgsConstructor;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.core.io.Resource;
import org.springframework.stereotype.Service;
import org.yaml.snakeyaml.Yaml;

import jakarta.annotation.PostConstruct;
import java.io.InputStream;
import java.util.*;

@Service
@RequiredArgsConstructor
public class SourceClassifierService {

    @Value("classpath:classification/header-signatures.yaml")
    private Resource signatureFile;

    private Map<SourceType, HeaderSignature> signatures;

    @PostConstruct
    public void load() {
        try (InputStream is = signatureFile.getInputStream()) {
            Yaml yaml = new Yaml();
            Map<String, Object> parsed = yaml.load(is);
            this.signatures = parseSignatures(parsed);
        } catch (Exception e) {
            throw new IllegalStateException("Failed to load classification rules", e);
        }
    }

    public ClassificationResult classify(ParsedWorkbook workbook) {
        // 1. 모든 시트의 처음 5 row를 헤더 후보로 추출
        Set<String> headerTokens = extractHeaderTokens(workbook);

        // 2. 파일명도 시그니처에 포함 (영업 부서 관행: 파일명에 출처 표기)
        headerTokens.add(workbook.filename());

        // 3. 각 SourceType에 대해 매칭 점수 계산
        Map<SourceType, Double> scores = new EnumMap<>(SourceType.class);
        for (var entry : signatures.entrySet()) {
            SourceType type = entry.getKey();
            HeaderSignature sig = entry.getValue();
            double score = computeScore(headerTokens, sig);
            scores.put(type, score);
        }

        // 4. 최고 점수가 임계치(0.5) 이상이면 그 타입으로 분류, 아니면 UNRECOGNIZED
        return scores.entrySet().stream()
            .max(Map.Entry.comparingByValue())
            .filter(e -> e.getValue() >= 0.5)
            .map(e -> new ClassificationResult(e.getKey(), e.getValue(), scores))
            .orElse(new ClassificationResult(SourceType.UNRECOGNIZED, 0.0, scores));
    }

    private Set<String> extractHeaderTokens(ParsedWorkbook workbook) {
        Set<String> tokens = new HashSet<>();
        for (ParsedSheet sheet : workbook.sheets()) {
            tokens.add(sheet.sheetName());
            sheet.rows().stream()
                .limit(5)                                  // 처음 5행만 헤더 후보
                .flatMap(row -> row.cells().stream())
                .filter(s -> s != null && !s.isBlank())
                .forEach(tokens::add);
        }
        return tokens;
    }

    private double computeScore(Set<String> tokens, HeaderSignature sig) {
        // 1. 제외 키워드가 있으면 0점
        boolean excluded = sig.excluded().stream()
            .anyMatch(kw -> tokens.stream()
                .anyMatch(t -> t.toUpperCase(Locale.ROOT)
                    .contains(kw.toUpperCase(Locale.ROOT))));
        if (excluded) return 0.0;

        // 2. 필수 키워드 (any) 중 매칭 개수
        long matches = sig.requiredAny().stream()
            .filter(kw -> tokens.stream()
                .anyMatch(t -> t.toUpperCase(Locale.ROOT)
                    .contains(kw.toUpperCase(Locale.ROOT))))
            .count();

        // 3. 매칭 개수 × 가중치, 최대 1.0 정규화
        return Math.min(1.0, matches * sig.weight() / sig.requiredAny().size());
    }
```

#### D. `HeaderSignature.java` & `ClassificationResult.java` — DTO

```java
package com.scheduling.order.parser;

import java.util.List;
import java.util.Map;

public record HeaderSignature(
    List<String> requiredAny,
    List<String> excluded,
    double weight
) {
    public HeaderSignature {
        if (requiredAny == null) requiredAny = List.of();
        if (excluded == null)     excluded    = List.of();
        if (weight <= 0)          weight      = 1.0;
    }
}

public record ClassificationResult(
    SourceType sourceType,
    double confidence,                   // 0.0 ~ 1.0
    Map<SourceType, Double> allScores    // 디버깅·근거 추적용
) {}
```

### 측정 가능 기준

- [ ] **30개 회귀 워크북 분류 정확도 ≥ 99%** (29/30 이상 정확) — TC-OC-002
- [ ] 4종 SourceType 각각 ≥ 7개 워크북 회귀 (총 30 = 8+8+7+7 권장)
- [ ] **UNRECOGNIZED 케이스** → false-positive 0건 (다른 타입을 UNRECOGNIZED로 오분류 X)
- [ ] `header-signatures.yaml` 변경 시 **재시작 없이 룰셋 reload** 가능 (Spring `@RefreshScope` 또는 핫리로드)
- [ ] 시그니처 매칭 알고리즘 **대소문자 무시·공백 정규화**
- [ ] 분류 결과의 `confidence` 값이 다음 기준 만족:
  - 정상 분류 → confidence ≥ 0.7
  - 모호 케이스 → 0.5 ≤ confidence < 0.7 (사용자에게 확인 요청)
  - 미식별 → confidence < 0.5

### 회귀 시나리오 (T-U + A)

- **케이스 1~7 (월별 예상)**: 워크북 7종 → 모두 MONTHLY_FORECAST

- **케이스 15~21 (확정)**: 워크북 7종 → 모두 CONFIRMED_ORDER

- **케이스 29 (모호)**: "월별 KD 예상" 같은 혼합 → 가장 강한 시그니처(KD) 우선
- **케이스 30 (미식별)**: 무관한 워크북 (예: "재고 현황") → UNRECOGNIZED
