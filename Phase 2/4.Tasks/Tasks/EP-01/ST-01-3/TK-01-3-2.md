#### C. `FileHashCalculator.java` — SHA-256

```java
package com.scheduling.order.watcher;

import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.io.IOException;
import java.nio.file.*;
import java.security.MessageDigest;
import java.security.NoSuchAlgorithmException;

@Slf4j
@Component
public class FileHashCalculator {

    public String sha256(Path file) {
        try (var is = Files.newInputStream(file)) {
            MessageDigest md = MessageDigest.getInstance("SHA-256");
            byte[] buf = new byte[8192];
            int read;
            while ((read = is.read(buf)) > 0) {
                md.update(buf, 0, read);
            }
            return toHex(md.digest());
        } catch (IOException | NoSuchAlgorithmException e) {
            throw new IllegalStateException("Hash failed: " + file, e);
        }
    }

    private String toHex(byte[] bytes) {
        StringBuilder sb = new StringBuilder(bytes.length * 2);
        for (byte b : bytes) sb.append(String.format("%02x", b));
        return sb.toString();
    }
}
```

#### D. `FileIngestQueueService.java` — 이벤트 핸들러 핵심

```java
package com.scheduling.order.watcher;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.context.event.EventListener;
import org.springframework.scheduling.annotation.Async;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.time.Instant;
import java.util.UUID;

@Slf4j
@Service
@RequiredArgsConstructor
public class FileIngestQueueService {

    private final PickedFileRepository repo;
    private final FileHashCalculator hasher;
    private final InternalImportClient importClient;

    /**
     * TK-01-3-1의 FileDiscoveredEvent 수신
     * - 중복 해시 검사 → SKIPPED_DUPLICATE 또는 PROCESSING
     * - 내부 Import API 호출 → tracking_id 받음
     * - audit row 기록 (모든 경로)
     */
    @Async("excelImportExecutor")
    @EventListener
    @Transactional
    public void handleFileDiscovered(FileDiscoveredEvent event) {
        var path = event.filePath();
        log.info("Picked file: {} (size={}, via={})", path, event.sizeBytes(), event.source());

        try {
            // 1. SHA-256 해시 계산 (파일 close 후 호출되므로 안정성 가정 — TK-01-3-3에서 close 검증)
            String hash = hasher.sha256(path);

            // 2. 같은 해시 + 최근 24h 내 중복 검사
            boolean isDuplicate = repo.existsByFileHashAndStatusInWithin(
                hash,
                List.of(PickedFileStatus.INGESTED, PickedFileStatus.PROCESSING),
                Instant.now().minusSeconds(24 * 3600)
            );

            // 3. QUEUED 행 INSERT (audit 필수)
            var picked = PickedFile.builder()
                .filePath(path.toString())
                .fileName(path.getFileName().toString())
                .fileHash(hash)
                .sizeBytes(event.sizeBytes())
                .discoveredAt(event.discoveredAt())
                .discoveredVia(event.source())
                .status(isDuplicate ? PickedFileStatus.SKIPPED_DUPLICATE : PickedFileStatus.QUEUED)
                .build();
            repo.save(picked);

            if (isDuplicate) {
                log.info("Skipping duplicate file: {} (hash={})", path, hash);
                return;
            }

            // 4. 내부 Import 호출
            picked.setStatus(PickedFileStatus.PROCESSING);
            repo.save(picked);

            UUID trackingId = importClient.importFile(path);
            picked.setTrackingId(trackingId);
            picked.setStatus(PickedFileStatus.INGESTED);
            picked.setProcessedAt(Instant.now());
            repo.save(picked);

            log.info("Ingested {} → trackingId {}", path, trackingId);

        } catch (Exception e) {
            log.error("Failed to ingest {}", path, e);
            // FAILED row 기록 — audit 정합
            repo.save(PickedFile.builder()
                .filePath(path.toString())
                .fileName(path.getFileName().toString())
                .fileHash("error:" + UUID.randomUUID())   // 임시 해시
                .sizeBytes(event.sizeBytes())
                .discoveredAt(event.discoveredAt())
                .discoveredVia(event.source())
                .status(PickedFileStatus.FAILED)
                .errorMessage(e.getMessage())
                .build());
            // 재시도 큐 등록 (선택 — Phase 2+ Spring Retry)
        }
    }
}
```

     * 시스템 actor("system:watcher")로 audit 기록.

        UUID trackingId = UUID.randomUUID();

  - JPA Enum 매핑은 `@Enumerated(EnumType.STRING)` 필수 (ORDINAL은 변경에 취약)
- **유사 패턴**:
  - 향후 자동 import 기능 추가 시 (예: MES 실적 자동 수신) 동일 패턴 차용
- **표준 참조**:
  - Spring Data JPA: https://docs.spring.io/spring-data/jpa/reference/jpa.html
  - Flyway Versioning: https://flywaydb.org/documentation/concepts/migrations#versioned-migrations
- **참고 외부 자료**:
  - "Event Listener with @TransactionalEventListener" — Spring 공식
