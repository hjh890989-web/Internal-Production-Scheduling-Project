#### C. `SchemaMappingService.java` — 핵심

```java
package com.scheduling.order.mapping;

import com.scheduling.order.domain.*;
import com.scheduling.order.parser.*;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import io.micrometer.core.instrument.MeterRegistry;
import org.springframework.stereotype.Service;

import java.util.*;

@Slf4j
@Service
@RequiredArgsConstructor
public class SchemaMappingService {

    private final MappingRuleLoader ruleLoader;
    private final FieldNormalizer normalizer;
    private final MeterRegistry meterRegistry;

    /**
     * ParsedWorkbook + ClassificationResult → MappingResult
     * - success: List<OrderDraft>
     * - failures: List<MappingFailure>
     */
    public MappingResult map(ParsedWorkbook workbook, SourceType sourceType) {
        MappingRule rule = ruleLoader.loadRuleFor(sourceType);

        List<OrderDraft> successes = new ArrayList<>();
        List<MappingFailure> failures = new ArrayList<>();

        for (ParsedSheet sheet : workbook.sheets()) {
            int headerRowIdx = detectHeaderRow(sheet, rule);
            Map<String, Integer> columnIndex = buildColumnIndex(sheet.rows().get(headerRowIdx), rule);

            // 헤더 다음 행부터 데이터
            for (int rowIdx = headerRowIdx + 1; rowIdx < sheet.rows().size(); rowIdx++) {
                ParsedRow row = sheet.rows().get(rowIdx);
                try {
                    OrderDraft draft = mapRow(row, columnIndex, rule);
                    successes.add(draft);
                } catch (MappingException e) {
                    failures.add(new MappingFailure(
                        sheet.sheetName(), rowIdx,
                        row.cells(),
                        e.getField(), e.getReason()
                    ));
                }
            }
        }

        // K-O03 메트릭 emit
        int total = successes.size() + failures.size();
        if (total > 0) {
            double successRate = (double) successes.size() / total;
            meterRegistry.gauge("order.mapping.success_rate",
                List.of(io.micrometer.core.instrument.Tag.of("source_type", sourceType.name())),
                successRate
            );
            meterRegistry.counter("order.mapping.rows.total",
                "source_type", sourceType.name()).increment(total);
            meterRegistry.counter("order.mapping.rows.failed",
                "source_type", sourceType.name()).increment(failures.size());
        }

        log.info("Mapping for {}: {}/{} success ({:.1f}%)",
            sourceType, successes.size(), total, 100.0 * successes.size() / Math.max(1, total));

        return new MappingResult(successes, failures, sourceType);
    }

    private int detectHeaderRow(ParsedSheet sheet, MappingRule rule) {
        for (int candidateIdx : rule.headerRowCandidates()) {
            if (candidateIdx >= sheet.rows().size()) continue;
            ParsedRow row = sheet.rows().get(candidateIdx);
            // 별칭 1개 이상 포함되어 있으면 헤더로 판정
            for (String cell : row.cells()) {
                if (rule.allAliases().contains(normalizer.canonical(cell))) {
                    return candidateIdx;
                }
            }
        }
        throw new MappingException("HEADER", "헤더 행을 찾을 수 없음");
    }

    private Map<String, Integer> buildColumnIndex(ParsedRow header, MappingRule rule) {
        Map<String, Integer> map = new HashMap<>();
        for (int i = 0; i < header.cells().size(); i++) {
            String canonical = normalizer.canonical(header.cells().get(i));
            // 별칭 → 표준 필드명
            String standardField = rule.resolveAliasToField(canonical);
            if (standardField != null) {
                map.put(standardField, i);
            }
        }
        return map;
    }

    private OrderDraft mapRow(ParsedRow row, Map<String, Integer> idx, MappingRule rule) {
        OrderDraft.Builder b = OrderDraft.builder();

        // hose_id — 필수, 정규식 검증
        b.hoseId(normalizer.normalizeHoseId(
            getCell(row, idx, "hose_id", true), rule));

        // delivery_date — 필수, 다중 포맷
        b.deliveryDate(normalizer.parseDate(
            getCell(row, idx, "delivery_date", true), rule));

        // qty — 필수, 정수 강제
        b.qty(normalizer.parseInt(
            getCell(row, idx, "qty", true), rule));

        // order_type — 룰 기본값
        b.orderType(rule.defaultOrderType());

        // customer — 선택, 기본값
        b.customer(Optional.ofNullable(getCell(row, idx, "customer", false))
            .filter(s -> !s.isBlank())
            .orElse(rule.defaultCustomer()));

        return b.build();
    }

    private String getCell(ParsedRow row, Map<String, Integer> idx, String field, boolean required) {
        Integer i = idx.get(field);
        if (i == null) {
            if (required) throw new MappingException(field, "표준 필드 컬럼 없음");
            return null;
        }
        if (i >= row.cells().size()) {
            if (required) throw new MappingException(field, "row 길이 부족");
            return null;
        }
        return row.cells().get(i);
    }
}
```

#### D. `OrderDraft.java` — 표준 스키마 DTO

```java
package com.scheduling.order.domain;

import jakarta.validation.constraints.*;
import lombok.Builder;

import java.time.LocalDate;
import java.util.UUID;

@Builder
public record OrderDraft(
    UUID orderId,                              // null이면 commit 시 발급
    @NotBlank @Pattern(regexp = "^[A-Z0-9-]+$") String hoseId,
    @NotNull LocalDate deliveryDate,
    @Min(1) int qty,
    @NotNull OrderType orderType,
    @NotBlank String customer
) {
    public OrderDraft {
        if (orderId == null) orderId = UUID.randomUUID();
        if (customer == null || customer.isBlank()) customer = "내수";
    }
}
```

#### E. `MappingFailure.java`

```java
package com.scheduling.order.mapping;

import java.util.List;

public record MappingFailure(
    String sheetName,
    int rowIndex,
    List<String> originalCells,
    String failedField,          // hose_id·delivery_date·qty·customer·HEADER
    String reason                // "포맷 인식 실패"·"필수 컬럼 없음"·"음수 수량"
) {
    public String userMessage() {
        return "%s 시트 %d행: %s — %s".formatted(
            sheetName, rowIndex + 1, failedField, reason);
    }
}
```

## :memo: Implementation Notes

- **기술 힌트**:
  - `Jakarta Validation`(`@NotBlank`·`@Pattern` 등)이 Spring Boot 3.x 자동 활성 — `MethodValidationPostProcessor` 불필요
  - `Micrometer Gauge`는 lambda보다 `MeterRegistry.gauge(name, tags, object, function)` 패턴이 안전 (GC 방지)
  - `LocalDate.parse(s, DateTimeFormatter)`로 다중 포맷 시도 — try 순서 중요
  - **알고리즘 결정성**: 같은 입력 → 같은 출력 (테스트 안정성)
  - **불변 객체**: `OrderDraft` 는 record로 불변 — 멀티스레드 안전
- **유사 패턴**:
  - 다른 도메인(PRODUCT·VC_CONSTRAINT) 마스터 데이터 import 시 동일 패턴 차용 가능
- **표준 참조**:
  - Jakarta Validation 3.0 (JSR-380): https://beanvalidation.org/3.0/
  - Spring Boot Externalized Configuration (§10): https://docs.spring.io/spring-boot/docs/3.3.x/reference/html/features.html#features.external-config
- **참고 외부 자료**:
  - "Bean Validation with Spring Boot" — Baeldung
