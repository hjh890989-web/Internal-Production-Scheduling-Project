### Flyway `V2_8__create_v_product_with_spec.sql`

```sql
CREATE OR REPLACE VIEW master.v_product_with_spec AS
SELECT
    vc.hose_id,
    ex.spec_value             AS spec,                    -- B열 규격
    vc.machine_id             AS preferred_machine,       -- VC 기본 호기
    vc.composite_count,
    vc.lp_left_setting,
    vc.lp_right_setting,
    ex.angle_count            AS angle_count,             -- 앵글 수
    (ex.spec_value < 7)       AS is_spec_lt7              -- BR-V17 marker
FROM master.vc_constraint vc
LEFT JOIN master.ex_constraint ex
    ON vc.hose_id = ex.hose_id;

COMMENT ON VIEW master.v_product_with_spec IS 'EP-21 ADR-017 cross-master: VC+EX 결합. BR-V17 spec<7 검증용.';

-- 인덱스는 underlying 테이블에 존재 — VIEW는 별도 불필요
-- 단, hose_id 조회 성능 보장 위해 EX_CONSTRAINT에 PK(hose_id) 확인
GRANT SELECT ON master.v_product_with_spec TO app_user;
```

### `ProductSpec.java`

```java
@Entity @Immutable @Subselect("SELECT * FROM master.v_product_with_spec")
@Synchronize({"master.vc_constraint", "master.ex_constraint"})
public class ProductSpec {
    @Id @Column(name="hose_id") private String hoseId;
    @Column(name="spec") private Integer spec;
    @Column(name="preferred_machine") private String preferredMachine;
    @Column(name="composite_count") private Integer compositeCount;
    @Column(name="lp_left_setting", length=1) private String lpLeftSetting;
    @Column(name="lp_right_setting", length=1) private String lpRightSetting;
    @Column(name="angle_count") private Integer angleCount;
    @Column(name="is_spec_lt7") private Boolean isSpecLt7;

    public boolean isSpecLt7() { return Boolean.TRUE.equals(isSpecLt7); }
}
```

### `ProductSpecRepository.java`

```java
public interface ProductSpecRepository extends Repository<ProductSpec, String> {
    Optional<ProductSpec> findById(String hoseId);
    List<ProductSpec> findAllByIsSpecLt7True();
}
```

## :memo: Implementation Notes

- `@Immutable @Subselect` Hibernate read-only 매핑 — 영속성 컨텍스트 외 변경 차단
- `@Synchronize` 양쪽 테이블 변경 시 캐시 강제 flush
- Materialized VIEW 미사용 — refresh latency vs read 빈도 트레이드오프 (ADR-017 결정)
