### `SpecLt7CapRule.java`

```java
@Component @RequiredArgsConstructor
public class SpecLt7CapRule {

    private static final int SPEC_LT7_MAX_ANGLES = 4;     // BR-V17

    private final ProductSpecCache productSpecCache;

    /**
     * BR-V17: 규격<7 품번의 가류기 1대 동시 점유 앵글 ≤ 4.
     * 같은 (machineId, date)에서 is_spec_lt7=TRUE 품번들의 angle_count 합산.
     */
    public RuleResult evaluate(SlotCandidate cand, AllocationLedger ledger) {
        var spec = productSpecCache.get(cand.hoseId()).orElse(null);
        if (spec == null || !spec.isSpecLt7()) return RuleResult.pass();

        int candAngles = spec.getAngleCount();
        int currentAngles = ledger.countAnglesForSpecLt7(cand.machineId(), cand.productionDate());

        if (currentAngles + candAngles > SPEC_LT7_MAX_ANGLES) {
            return RuleResult.fail(
                "BR-V17 spec<7 가류기당 ≤4 위반: %s × %s/%s spec=%d 현재 %d + 계획 %d > %d"
                .formatted(cand.hoseId(), cand.machineId(), cand.productionDate(),
                            spec.getSpec(), currentAngles, candAngles, SPEC_LT7_MAX_ANGLES));
        }

        return RuleResult.pass();
    }
}
```

### `AllocationLedger.java` 수정

```java
public class AllocationLedger {
    // ... 기존 slotCounts

    // (machine, date) → 누적 angle 수 (spec<7 품번만)
    private final Map<Tuple2<String, LocalDate>, Integer> specLt7Angles = new HashMap<>();

    public int countAnglesForSpecLt7(String machineId, LocalDate date) {
        return specLt7Angles.getOrDefault(Tuple.of(machineId, date), 0);
    }

    public void addAnglesForSpecLt7(String machineId, LocalDate date, int angles) {
        var key = Tuple.of(machineId, date);
        specLt7Angles.merge(key, angles, Integer::sum);
    }
}
```

### `RulePipeline.java` 통합

```java
@Component @RequiredArgsConstructor
public class RulePipeline {

    private final SlotOxRule slotOxRule;
    private final MachinePinRule machinePinRule;
    private final LeftRightRule leftRightRule;
    private final HoseSlotCapRule hoseSlotCapRule;
    private final SpecLt7CapRule specLt7CapRule;             // ← 추가

    public RuleResult evaluate(SlotCandidate cand, AllocationLedger ledger) {
        var r1 = slotOxRule.evaluate(cand);             if (!r1.passed()) return r1;
        var r2 = machinePinRule.evaluate(cand);         if (!r2.passed()) return r2;
        var r3 = leftRightRule.evaluate(cand);          if (!r3.passed()) return r3;
        var r4 = hoseSlotCapRule.evaluate(cand, ledger); if (!r4.passed()) return r4;
        return specLt7CapRule.evaluate(cand, ledger);   // ← 마지막
    }
}
```
