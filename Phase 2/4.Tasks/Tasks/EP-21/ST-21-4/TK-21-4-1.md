### `HoseSlotCapRule.java`

```java
@Component @RequiredArgsConstructor
public class HoseSlotCapRule {

    private final HoseRuleCache hoseRuleCache;
    private final ConcurrentSlotsRule concurrentRule;

    /**
     * BR-V15·V16: side_lock 필터 후 회전당 ≤ max_concurrent_slots.
     * - 28422-2M800 (side_lock=RIGHT, max=2): 우측 슬롯만 카운트, 회전당 ≤ 2
     * - 28421-2M800 (side_lock=LEFT,  max=2): 좌측 슬롯만 카운트, 회전당 ≤ 2
     */
    public RuleResult evaluate(SlotCandidate cand, AllocationLedger ledger) {
        var ruleOpt = hoseRuleCache.get(cand.hoseId());
        if (ruleOpt.isEmpty()) return RuleResult.pass();

        var rule = ruleOpt.get();
        SlotSide candSide = SlotSide.of(cand.machineType(), cand.slotNumber());

        // 1) side_lock 검증 (TK-21-1-2 LeftRightRule이 이미 처리하지만 결합 시 명시)
        if (rule.hasSideLock() && rule.getSideLock() != candSide) {
            return RuleResult.fail("BR-V15·V16 side_lock 위반: %s × side=%s (lock=%s)"
                .formatted(cand.hoseId(), candSide, rule.getSideLock()));
        }

        // 2) max_concurrent_slots — side_lock 적용 슬롯만 카운트
        int current = ledger.countSlots(cand.hoseId(), cand.productionDate(),
                                         cand.machineId(), cand.rotationId());
        if (current + 1 > rule.getMaxConcurrentSlots()) {
            return RuleResult.fail("BR-V15·V16 회전당 슬롯 초과: %s × %s/%s/%s = %d > %d"
                .formatted(cand.hoseId(), cand.machineId(), cand.productionDate(),
                            cand.rotationId(), current + 1, rule.getMaxConcurrentSlots()));
        }

        return RuleResult.pass();
    }
}
```
