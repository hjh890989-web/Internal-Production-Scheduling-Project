### `RulePipeline.java`

```java
@Component @RequiredArgsConstructor
public class RulePipeline {

    private final SlotOxRule slotOxRule;              // EP-04 (BR-V13)
    private final MachinePinRule machinePinRule;     // ST-21-3 (BR-V14)
    private final LeftRightRule leftRightRule;        // ST-21-1 (BR-V15·V16)
    private final HoseSlotCapRule hoseSlotCapRule;   // ST-21-4 (결합)

    /**
     * Allocator가 후보별로 호출. 순서: 빠른 fail-first.
     * 1. SlotOxRule (마스터 lookup, O(1))
     * 2. MachinePinRule (캐시 lookup, O(1)) — 머신 사전 필터에서 이미 처리, 방어
     * 3. LeftRightRule (캐시 lookup, O(1))
     * 4. HoseSlotCapRule (ledger lookup, O(1))
     */
    public RuleResult evaluate(SlotCandidate cand, AllocationLedger ledger) {
        var r1 = slotOxRule.evaluate(cand);             if (!r1.passed()) return r1;
        var r2 = machinePinRule.evaluate(cand);         if (!r2.passed()) return r2;
        var r3 = leftRightRule.evaluate(cand);          if (!r3.passed()) return r3;
        var r4 = hoseSlotCapRule.evaluate(cand, ledger);
        return r4;
    }
}
```

### `GreedyRotationAllocator` 수정

```java
@Service @RequiredArgsConstructor
public class GreedyRotationAllocator {

    private final RulePipeline pipeline;
    private final AllocationLedger ledger;
    // ...

    public AllocationResult allocate(AllocationContext ctx) {
        // ... 1주 호라이즌 순회
        for (LocalDate date : ctx.workingDays()) {
            for (Machine machine : candidateMachines(hose, allMachines)) {     // machine_pin 사전 필터
                for (RotationId rotation : machine.rotations(date)) {
                    for (int slot = 1; slot <= machine.slotCount(); slot++) {
                        var cand = SlotCandidate.of(hose, date, machine, rotation, slot);
                        var result = pipeline.evaluate(cand, ledger);
                        if (!result.passed()) {
                            conflicts.add(AllocationConflict.from(cand, result));
                            continue;
                        }
                        // 배치 확정
                        schedule.add(cand);
                        ledger.addSlots(hose, date, machine.id(), rotation, 1);
                        // cumulativeYield 갱신 ...
                    }
                }
            }
        }
    }
}
```
