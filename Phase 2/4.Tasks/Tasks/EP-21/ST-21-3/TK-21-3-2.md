### `ConcurrentSlotsRule.java`

```java
@Component @RequiredArgsConstructor
public class ConcurrentSlotsRule {

    private final HoseRuleCache hoseRuleCache;

    /**
     * BR-V14·V15·V16: 같은 (hose, date, machine, rotation) 내 동시 점유 슬롯 ≤ max_concurrent_slots.
     * 회전 단위 검증 (주간/야간 분리).
     */
    public boolean validate(String hoseId, LocalDate date, String machineId, RotationId rotation,
                            int plannedSlots, AllocationLedger ledger) {
        int max = hoseRuleCache.get(hoseId)
            .map(VcHoseRule::getMaxConcurrentSlots)
            .orElse(99);

        int current = ledger.countSlots(hoseId, date, machineId, rotation);
        return (current + plannedSlots) <= max;
    }

    public RuleResult evaluate(String hoseId, LocalDate date, String machineId,
                                RotationId rotation, int plannedSlots, AllocationLedger ledger) {
        if (!validate(hoseId, date, machineId, rotation, plannedSlots, ledger)) {
            int max = hoseRuleCache.get(hoseId)
                .map(VcHoseRule::getMaxConcurrentSlots).orElse(99);
            return RuleResult.fail("BR-V14 동시 슬롯 초과: %s × %s/%s/%s 현재+계획=%d > %d"
                .formatted(hoseId, machineId, date, rotation, ledger.countSlots(hoseId, date, machineId, rotation) + plannedSlots, max));
        }
        return RuleResult.pass();
    }
}
```

### `AllocationLedger.java` 수정

```java
public class AllocationLedger {

    // (hose, date, machine, rotation) → slot count
    private final Map<LedgerKey, Integer> slotCounts = new HashMap<>();

    public int countSlots(String hoseId, LocalDate date, String machineId, RotationId rotation) {
        return slotCounts.getOrDefault(new LedgerKey(hoseId, date, machineId, rotation), 0);
    }

    public void addSlots(String hoseId, LocalDate date, String machineId, RotationId rotation, int count) {
        var key = new LedgerKey(hoseId, date, machineId, rotation);
        slotCounts.merge(key, count, Integer::sum);
    }

    public record LedgerKey(String hoseId, LocalDate date, String machineId, RotationId rotation) {}
}
```
