### `MachinePinRule.java`

```java
@Component @RequiredArgsConstructor
public class MachinePinRule implements VcRule {

    private final HoseRuleCache hoseRuleCache;

    /**
     * BR-V14: VC_HOSE_RULE.machine_pin 강제.
     *  - machine_pin = NULL → 자유 배치 (pass)
     *  - machine_pin = 'LP-01' → machineId == 'LP-01' 만 pass
     *  - lp_only = TRUE → IC 호기 거부
     */
    public boolean validate(String hoseId, String machineId, MachineType type) {
        var ruleOpt = hoseRuleCache.get(hoseId);
        if (ruleOpt.isEmpty()) return true;     // 룰 없음 = 자유

        var rule = ruleOpt.get();

        // 1) lp_only 검증
        if (Boolean.TRUE.equals(rule.getLpOnly()) && type == MachineType.IC) {
            return false;
        }

        // 2) machine_pin 검증
        if (rule.hasMachinePin() && !rule.getMachinePin().equals(machineId)) {
            return false;
        }

        return true;
    }

    @Override
    public RuleResult evaluate(SlotCandidate cand) {
        if (!validate(cand.hoseId(), cand.machineId(), cand.machineType())) {
            return RuleResult.fail("BR-V14 machine_pin/lp_only 위반: %s × %s"
                .formatted(cand.hoseId(), cand.machineId()));
        }
        return RuleResult.pass();
    }
}
```

### `GreedyRotationAllocator` 후보 필터 통합

```java
// 머신 순회 단계에서 사전 필터링 (성능)
private List<Machine> candidateMachines(String hoseId, List<Machine> all) {
    var rule = hoseRuleCache.get(hoseId).orElse(null);
    return all.stream()
        .filter(m -> machinePinRule.validate(hoseId, m.id(), m.type()))
        .toList();
}
```
