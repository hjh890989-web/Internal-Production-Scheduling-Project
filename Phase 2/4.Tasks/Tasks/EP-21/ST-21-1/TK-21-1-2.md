### 핵심 로직

#### A. `SlotSide.java`

```java
public enum SlotSide {
    LEFT, RIGHT;

    /**
     * 가류기 슬롯 번호 → 좌/우 매핑 (LP 8슬롯: 1~4 좌, 5~8 우 / IC 6슬롯: 1~3 좌, 4~6 우)
     */
    public static SlotSide of(MachineType type, int slotNumber) {
        return switch (type) {
            case LP -> slotNumber <= 4 ? LEFT : RIGHT;
            case IC -> slotNumber <= 3 ? LEFT : RIGHT;
        };
    }
}
```

#### B. `LeftRightRule.java`

```java
@Component @RequiredArgsConstructor
public class LeftRightRule implements VcRule {

    private final VcConstraintRepository repo;

    /**
     * BR-V15·V16: 슬롯 셋팅 방향이 품번의 lp_left/right_setting과 일치해야 함.
     * lp_left_setting = 'O' → LEFT 슬롯 허용
     * lp_right_setting = 'O' → RIGHT 슬롯 허용
     */
    public boolean validate(String hoseId, String machineId, SlotSide side) {
        var constraint = repo.find(hoseId, machineId)
            .orElseThrow(() -> new MasterNotFoundException(hoseId + "/" + machineId));
        return switch (side) {
            case LEFT  -> constraint.allowsLeft();
            case RIGHT -> constraint.allowsRight();
        };
    }

    @Override
    public RuleResult evaluate(SlotCandidate cand) {
        SlotSide side = SlotSide.of(cand.machineType(), cand.slotNumber());
        if (!validate(cand.hoseId(), cand.machineId(), side)) {
            return RuleResult.fail("BR-V15·V16 좌/우 셋팅 불일치: %s × %s slot=%d(%s)"
                .formatted(cand.hoseId(), cand.machineId(), cand.slotNumber(), side));
        }
        return RuleResult.pass();
    }
}
```
