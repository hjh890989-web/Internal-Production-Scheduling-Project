### 핵심 로직

#### A. `MasterCompatController.java`

```java
package com.scheduling.master.api;

import com.scheduling.master.vc.SlotCompatibilityMatrix;
import com.scheduling.master.vc.SlotCompatibilityMatrixService;
import lombok.RequiredArgsConstructor;
import org.springframework.http.*;
import org.springframework.security.access.prepost.PreAuthorize;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/v1/master/compat")
@RequiredArgsConstructor
public class MasterCompatController {

    private final SlotCompatibilityMatrixService matrixService;

    /**
     * 슬롯 적합성 매트릭스 전체 조회.
     * - ETag = "v{version}" → 클라이언트 캐싱 (304 Not Modified)
     * - 응답 ≤ 500ms p95 (TC-VC-001 일부)
     */
    @GetMapping
    @PreAuthorize("isAuthenticated()")
    public ResponseEntity<CompatibilityResponse> get(
        @RequestHeader(value = HttpHeaders.IF_NONE_MATCH, required = false) String ifNoneMatch
    ) {
        SlotCompatibilityMatrix matrix = matrixService.current();
        String etag = "\"v" + matrix.getVersion() + "\"";

        if (etag.equals(ifNoneMatch)) {
            return ResponseEntity.status(HttpStatus.NOT_MODIFIED).eTag(etag).build();
        }

        return ResponseEntity.ok()
            .eTag(etag)
            .cacheControl(CacheControl.maxAge(java.time.Duration.ofMinutes(5))
                .mustRevalidate())
            .body(CompatibilityResponse.from(matrix));
    }

    /**
     * 특정 (품번, 슬롯) 조회 — 가벼운 응답.
     */
    @GetMapping("/{hoseId}/{slotPosition}")
    @PreAuthorize("isAuthenticated()")
    public CompatibilityResponse.PointCheck check(
        @PathVariable String hoseId,
        @PathVariable String slotPosition
    ) {
        SlotCompatibilityMatrix matrix = matrixService.current();
        var slot = com.scheduling.master.vc.SlotPosition.valueOf(slotPosition);
        return new CompatibilityResponse.PointCheck(
            hoseId, slotPosition, matrix.isEligible(hoseId, slot)
        );
    }
}
```

#### B. `CompatibilityResponse.java`

```java
package com.scheduling.master.api;

import com.scheduling.master.vc.SlotCompatibilityMatrix;
import com.scheduling.master.vc.SlotPosition;

import java.time.Instant;
import java.util.Map;
import java.util.Set;

public record CompatibilityResponse(
    int version,
    Instant builtAt,
    Map<String, Map<SlotPosition, Boolean>> matrix,        // byHose
    Map<SlotPosition, Set<String>> bySlot,
    Set<String> unschedulableHoseIds
) {
    public static CompatibilityResponse from(SlotCompatibilityMatrix m) {
        return new CompatibilityResponse(
            m.getVersion(),
            m.getBuiltAt(),
            m.getByHose(),
            m.getBySlot(),
            m.getUnschedulableHoseIds()
        );
    }

    public record PointCheck(String hoseId, String slotPosition, boolean eligible) {}
}
```

## :test_tube: Acceptance Criteria

**검증**: T-I + I

- [ ] `GET /api/v1/master/compat` 200 + 매트릭스 전체 응답
- [ ] **응답 시간 p95 ≤ 500ms** (NFR-PER-005 일부)
- [ ] **ETag 헤더** "v{version}" 형식
- [ ] **If-None-Match 일치 시 304 Not Modified** (Content-Length 0)
- [ ] **Cache-Control: max-age=300** (5분)
- [ ] `GET /api/v1/master/compat/{hoseId}/{slotPosition}` 단일 조회 응답

- [ ] 잘못된 slotPosition (예: `XYZ`) → 400 + 한국어 사유

## :memo: Implementation Notes

- ETag 패턴 — 매트릭스 version 기반이므로 결정적·캐시 친화적
- `Cache-Control: max-age=300` — 5분 클라이언트 캐시 + ETag 검증
- 응답 크기: 47품번 × 7 슬롯 × bool = 약 5KB JSON — 압축 시 ≤2KB
- 후속 ST-04-3 FE는 매트릭스를 IndexedDB 또는 Zustand persist로 영속 (오프라인 시 마지막 매트릭스 사용)
