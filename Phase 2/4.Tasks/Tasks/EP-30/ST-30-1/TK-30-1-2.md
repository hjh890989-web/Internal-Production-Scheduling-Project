```json
  "alias": "corporate-saml",

  "enabled": true,
  "trustEmail": true,

  "addReadTokenRoleOnCreate": false,
  "authenticateByDefault": false,
  "linkOnly": false,
  "firstBrokerLoginFlowAlias": "first broker login",
  "config": {
    "singleSignOnServiceUrl": "https://idp.company.com/SSO/SAML",
    "validateSignature": "true",
    "signingCertificate": "MIIC...",
    "principalAttribute": "email",
    "wantAssertionsSigned": "true",
    "syncMode": "FORCE"
  }
}
```

### Role Mapper (group → realm role)

```json
{
  "name": "AD Group → Realm Role",
  "identityProviderMapper": "saml-role-idp-mapper",
  "config": {
    "attribute.name": "memberOf",
    "attribute.value.regex": "CN=Scheduling-Planner,.*",
    "role": "PLANNER"
  }
}
```
