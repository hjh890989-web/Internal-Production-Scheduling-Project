## 7. BPMN Diagram

> Descriptive Conformance: Task / Sub-Process / Start·End Event / 기본 Gateway(X, +, O) / Sequence Flow / Message Flow / Data Object 까지만 사용.

### 7.1 As-Is (현행)

```mermaid
flowchart LR
    subgraph "Pool: 사내"
        subgraph "Lane: L1"
            S1((Start)) --> A1[Task 1]
        end
        subgraph "Lane: L2"
            A1 --> A2[Task 2]
            A2 --> E1((End))
        end
    end
```

### 7.2 To-Be (목표)

```mermaid
flowchart LR
    subgraph "Pool: 사내"
        subgraph "Lane: 시스템"
            S2((Start)) --> T1[Service Task]
        end
        subgraph "Lane: 사용자"
            T1 --> G1{Gateway X}
            G1 -->|조건| T2[User Task]
            G1 -->|기본| E2((End))
        end
    end
```

> 별도 `.bpmn` 정식 파일은 `/diagrams/PDD-XX.bpmn` 으로 별첨(향후 작성).
