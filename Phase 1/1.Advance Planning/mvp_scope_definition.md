# MVP 범위 정의서: 생산 계획 최적화 시스템

## 1. 프로젝트 목표
전체 공정 중 핵심인 **수주-성형-압출** 연동 시스템을 먼저 구축하여, 엑셀 기반 관리의 한계를 극복하고 계획의 정확도를 검증함.

## 2. 서비스 범위 (In-Scope)

### A. 수주 정보 통합 (Order Integration)
- **입력**: 월별 예상, KD, 주간 발주 엑셀 파일
- **기능**: 
  - 품번 기반 데이터 정합성 검사
  - 중복 데이터 필터링 및 통합 DB 구축
  - 제품군별 생산 대기 현황 대시보드

### B. 성형 공정 스케줄링 (Molding Scheduling)
- **목표**: 납품 예정일 2일 전 생산 완료
- **제약 조건**: 설비(프레스) 가동 능력, 금형 할당, 작업 시간 등
- **기능**: 
  - 납기 역산 기반 자동 일정 제안
  - 간트차트를 통한 수동 일정 조정 (Drag & Drop)

### C. 압출 공정 스케줄링 (Extrusion Scheduling)
- **목표**: 성형 공정 투입 1일 전 생산 완료
- **연동**: 확정된 성형 스케줄에서 관체 소요량 및 투입 시점 자동 도출
- **기능**: 
  - 성형 일정 변경 시 압출 일정 자동 재계산 (연동형 역산)

## 3. 제외 범위 (Out-of-Scope / Phase 2+)
- **자재 관리**: BOM 기반 소요량 계산(MRP), 자재 발주/입고 관리
- **외주 관리**: 조립 공정 등 외부 협력사 연동
- **기타 제품군**: 초기 선정된 파일럿 제품군 외의 데이터

## 4. 데이터 연동 (MES)
- **실적 수집**: 자체 개발 MES로부터 성형/압출 공정의 실제 생산 실적을 수집하여 계획 대비 진척도 표시
